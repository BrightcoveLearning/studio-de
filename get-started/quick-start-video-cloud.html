Nachdem Sie alle Themen in dieser Kurzanleitung überprüft haben, sollten Sie in der Lage sein:

  • Navigieren Sie in Video Cloud Studio
  • Laden Sie Videos in die Video Cloud hoch
  • Bearbeiten Sie die Videoeigenschaften
  • Erstellen und gestalten Sie einen einzelnen Videoplayer
  • Veröffentlichen Sie ein Video im Web
  • Erstellen Sie Wiedergabelisten
  • Erstellen Sie einen Playlist-Player
  • Veröffentlichen Sie eine Wiedergabeliste im Web

Zielgruppe

Jeder, der Video Cloud evaluiert oder neu in Video Cloud ist.

Voraussetzungen

  • Ein Video Cloud-Konto - Sie können sich für ein Testkonto anmelden Hier.
  • Sie sollten ein Beispielvideo haben, das Sie verwenden können oder können herunterladen Einige Beispielvideos

Gliederung

Jedes Thema in dieser Gliederung behandelt einen bestimmten Teil des Video-Publishing-Workflows mit Video Cloud. Jedes Thema verfügt über ein Übersichtsvideo, gefolgt von praktischen Schritten, denen Sie folgen können. Wenn Sie es vorziehen, nur alle Trainingsvideos anzusehen und keine der praktischen Übungen zu absolvieren, lesen Sie die Video Cloud-Grundlagen On-Demand-Schulung.

  • Ein Video hochladen
  • Bearbeiten grundlegender Videoeigenschaften
  • Spieler erstellen
  • Styling eines Players
  • Veröffentlichen eines einzelnen Videos im Web
  • Erstellen einer manuellen Wiedergabeliste
  • Erstellen einer intelligenten Wiedergabeliste
  • Erstellen eines Playlist-Players
  • Veröffentlichen eines Playlist-Players

Das Video Cloud Analytics-Modul wird im Rahmen dieses Schnellstarts nicht behandelt. Weitere Informationen zur Video Cloud-Analyse finden Sie im Einführung in Video Cloud Analytics On-Demand-Schulung.

Was soll ich als nächstes tun?

  • Für einen tieferen Einblick in die Video Cloud schauen Sie sich die an Einführung in die Video Cloud On-Demand-Schulung.
  • Informationen zu den verfügbaren Videoanalysetypen finden Sie unter Einführung in Video Cloud Analytics On-Demand-Schulung.
  • Informationen zum Veröffentlichen von Videoportalen oder In-Page-Erlebnissen mithilfe der Galerie finden Sie in der Erstellen von Portal-Erfahrungen mithilfe der Galerie oder Erstellen von In-Page-Erfahrungen mithilfe der Galerie On-Demand-Schulung.
