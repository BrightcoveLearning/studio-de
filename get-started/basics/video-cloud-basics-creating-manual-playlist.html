Voraussetzungen

  • Du hast hochgeladene Videos zu Ihrem Video Cloud-Konto

Schritte

  1. Öffne das Medien Modul.
  2. Klicken Sie im seitlichen Navigationsbereich auf Playlist erstellen Verknüpfung.
  3. Stellen Sie die Typ der Wiedergabeliste zu Handbuch.
  4. Geben Sie ein Name der Wiedergabeliste.
  5. Prüfen Wiedergabeliste in der Seitenleiste anzeigen Damit die Wiedergabeliste in der Seitenleiste angezeigt wird.
  6. Klicken speichern.
  7. Klicken Sie auf, um Videos zur Wiedergabeliste hinzuzufügen Alle Videos im seitlichen Navigationsbereich, um alle Videos im Konto anzuzeigen.
  8. Aktivieren Sie das Kontrollkästchen neben den Videos, die Sie zur Wiedergabeliste hinzufügen möchten.
  9. Klicken Mehr> Zur Wiedergabeliste hinzufügen.
  10. Wählen Sie eine Wiedergabeliste aus und klicken Sie dann auf Hinzufügen.

Um die Reihenfolge der Videos neu anzuordnen, wählen Sie die Wiedergabeliste aus, klicken Sie dann auf eine Zeile und ziehen sie an eine neue Position in der Liste.
