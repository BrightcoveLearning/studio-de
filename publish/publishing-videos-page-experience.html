In-Page-Erlebnisse werden mithilfe des Galerie-Moduls erstellt und ermöglichen die Veröffentlichung eines oder mehrerer Videos oder einer Wiedergabeliste auf einer Seite. Die Videos können in Karussells, Rastern und Wiedergabelisten angezeigt werden, die sich vor, während und nach der Videowiedergabe dynamisch ändern. Interaktive Elemente können sowohl innerhalb als auch außerhalb des Videoplayers hinzugefügt werden. Diese In-Page-Erlebnisse können dann problemlos in Websites, Zielseiten, Blogs und E-Mail-Kampagnen eingebettet werden.

Hinweis: Sie müssen Zugriff auf das Galerie-Modul haben, um In-Page-Erlebnisse veröffentlichen zu können.
Hinweis: Eine Übersicht über die Arbeit mit In-Page-Erfahrungen finden Sie unter Überblick: Erstellen, Bearbeiten und Veröffentlichen von In-Page-Erlebnissen.

Mit dem Medienmodul können Sie ein oder mehrere Videos oder eine Wiedergabeliste auswählen und anschließend eine In-Page-Erfahrung erstellen, die mithilfe des Galerie-Moduls angepasst und veröffentlicht werden kann.

Befolgen Sie diese Schritte, um mithilfe des Medienmoduls eine In-Page-Erfahrung zu erstellen.

  1. Öffne das Medien Modul.
  2. Wählen Sie die Videos oder Wiedergabelisten aus, die für die Erfahrung verwendet werden sollen. Um Videos auszuwählen, aktivieren Sie das Kontrollkästchen neben den Videos.

    Klicken Sie auf, um eine Wiedergabeliste auszuwählen Alle Wiedergabelisten Klicken Sie in der linken Navigation auf das Kontrollkästchen neben der Wiedergabeliste.

  3. Klicken Veröffentlichen und Einbetten> Galerie-In-Page-Erfahrung.
  4. Wählen Sie eine Vorlage aus, die für die Erfahrung verwendet werden soll. In diesem Beispiel ist die Gitter Vorlage ist ausgewählt.

    Informationen zu den In-Page-Vorlagen und einige Beispiele finden Sie unter Übersicht über In-Page-Erlebnisvorlagen für die Galerie.

    Hinweis: Sobald eine Vorlage ausgewählt ist, kann sie nicht geändert werden.
    Hinweis: Einige Vorlagen sind möglicherweise nicht verfügbar, wenn eine Wiedergabeliste oder mehrere Videos ausgewählt sind.
  5. Klicken Nächster.
  6. Geben Sie einen Erfahrungstitel und eine Beschreibung ein und wählen Sie eine Layoutgröße aus. Diese Werte können später geändert werden. Informationen zu diesen Einstellungen und ihrer Verwendung finden Sie unter Konfigurieren der Details für eine In-Page-Erfahrung.
  7. Klicken Erstellen. Der In-Page Experience-Editor wird in einem neuen Browser-Tab geöffnet.
