Nachdem Sie die Schritte in diesem Thema durchgearbeitet haben, sollten Sie in der Lage sein:

  • Navigieren Sie im Players-Modul
  • Erstelle einen Spieler
  • Legen Sie die Spielereigenschaften fest
  • Übergeben Sie eine Video Cloud-Video-ID, die im Player angezeigt werden soll
  • Vorschau und Veröffentlichung eines Players
Hinweis: Wenn Sie eine Wiedergabeliste oder ein einzelnes Video auf einem Player veröffentlichen möchten, verwenden Sie die Veröffentlichungsfunktion im Medienmodul. Weitere Informationen finden Sie unter Veröffentlichen von Videos und Wiedergabelisten mit dem Medienmodul.

Zielgruppe

Jeder Video-Publisher, der Videos mit dem Players-Modul erstellen und veröffentlichen muss.

Voraussetzungen

  • Ein Video Cloud-Konto mit Zugriff auf das Players-Modul
  • Ein Video in Ihrem Video Cloud-Konto, das veröffentlicht werden soll. Das in diesem Schnellstart verwendete Video kann heruntergeladen werden Hier oder Sie können Ihren eigenen Inhalt verwenden
  • Wenn Sie das mit diesem Schnellstart bereitgestellte Video verwenden, laden Sie es in Ihr Video Cloud-Konto hoch

Anleitung

Um einen Brightcove-Player mit Video Cloud Studio zu erstellen und zu veröffentlichen, führen Sie die folgenden Aufgaben aus:

  • Erstelle einen Spieler
  • Konfigurieren Sie die Player-Abmessungen
  • Veröffentlichen Sie den Player und zeigen Sie eine Vorschau an
  • Weisen Sie dem Player dynamisch Videoinhalte zu
  • Konfigurieren Sie die Stileigenschaften
  • Konfigurieren Sie Social-Sharing-Eigenschaften
  • Vorschau des Players
  • Veröffentlichen Sie den endgültigen Spieler

Erstelle einen Spieler

Führen Sie die folgenden Schritte aus, um einen neuen Player zu erstellen.

  1. Melden Sie sich bei Video Cloud Studio an.
  2. Der aktuelle Name des Video Cloud-Kontos wird in der oberen rechten Ecke der Seite angezeigt. Wenn Sie mehrere Konten haben, klicken Sie auf aktueller Benutzer - aktuelles Konto Verknüpfen Sie das Konto, in dem Sie den Player erstellen möchten, und wählen Sie es aus.
  3. Drücke den Spieler Link im Navigationsheader.
  4. Drücke den Neuer Spieler Taste.
  5. Geben Sie ein Name und kurze Beschreibung für den Spieler. Für diesen Schnellstart lautet der Spielername Beispiel-Player.
  6. Bestätigen Sie, dass der Spieler in der Liste der Spieler angezeigt wird.

Wenn ein neuer Player mit dem Players-Modul erstellt wird, wird der Player mit der neuesten Player-Version erstellt, die von Brightcove veröffentlicht wurde. Um die Player-Version anzuzeigen, öffnen Sie das Players-Modul und klicken Sie auf den Player, den Sie überprüfen möchten. In den Eigenschaften für Player-Informationen wird das angezeigt Veröffentlichte Player-Version.

Beachten Sie, dass möglicherweise nicht alle Spieler dieselbe Version haben. Während Brightcove den Brightcove-Player weiterentwickelt und verbessert, werden neue Player-Versionen veröffentlicht. Standardmäßig werden Updates automatisch auf alle Spieler angewendet. Um die automatische Aktualisierung eines Players zu verhindern, müssen Sie die Einstellungen vornehmen Aktualisierungsmodus Einstellung auf Handbuch (auch als Sperren eines Spielers bekannt). Weitere Informationen finden Sie unter Einstellen des Player-Aktualisierungsmodus und der Version.

Konfigurieren Sie die Player-Abmessungen

Durch Hinzufügen von Player-Abmessungen werden dem iframe-Einbettungscode Parameter für Breite und Höhe hinzugefügt. Wenn Sie die Abmessungen leer lassen, kann der Player in eine gelegt werden <div> von beliebiger Größe auf einer Seite und es wird erweitert, um den Raum zu füllen. Befolgen Sie diese Schritte, um die Player-Abmessungen festzulegen.

  1. Klicken Sie auf den Link für die Beispiel-Player um die Player-Eigenschaftenseite zu öffnen.
  2. Klicken Spielerinformationen im linken Navigationsmenü.
  3. Stellen Sie die Seitenverhältnis zu 16: 9.
  4. Stellen Sie die Spielerbreite zu 640.
  5. Klicken speichern.

Weitere Informationen zu den Eigenschaften der Player-Informationen finden Sie unter Festlegen der Eigenschaften der Player-Informationen.

Veröffentlichen Sie den Player und zeigen Sie eine Vorschau an

Durch das Veröffentlichen des Players werden die Änderungen auf den veröffentlichten Player übertragen. Weitere Informationen zu Vorschau-Playern, veröffentlichten Playern und Player-Veröffentlichungscode finden Sie unter Player-Einbettung generieren Code.

Führen Sie die folgenden Schritte aus, um den Player zu veröffentlichen.

  1. Klicken Veröffentlichen & Einbetten und dann Änderungen veröffentlichen um den Player zu veröffentlichen.
  2. Bestätigen Sie, dass die Veröffentlichung erfolgreich war.
  3. Drücke den Spieler-URL um den Vorschau-Player in einem neuen Browser-Tab zu öffnen. Der Player ist leer, da dem Player kein Video zugewiesen wurde.

Weisen Sie dem Player dynamisch Videoinhalte zu

Für diesen Schnellstart weisen wir dem Player mithilfe eines URL-Parameters dynamisch eine Video Cloud-Video-ID zu. Befolgen Sie dazu die folgenden Schritte.

  1. Öffnen Sie das Medienmodul und suchen Sie das Video, das Sie veröffentlichen möchten. Die Video-ID wird unter dem Videonamen angezeigt.
  2. Kopieren Sie die Video-ID in die Zwischenablage.
  3. Kehren Sie mit dem Preview-Player zur Registerkarte zurück. Um die anzuzeigende Video-ID dynamisch zu übergeben, fügen Sie Folgendes an die URL an und drücken Sie die Eingabetaste:
    ?videoId=your video id
    Notiz: Der Name des URL-Parameters berücksichtigt Groß- und Klein

    Die vollständige URL sieht ungefähr so aus:

    https://players.brightcove.net/1486906377/Ctzra7vEL_default/index.html?videoId=5798846643001
  4. Bestätigen Sie, dass das Video angezeigt wird.

Konfigurieren Sie die Stileigenschaften

Die Stileigenschaften werden verwendet, um das Erscheinungsbild des Players anzupassen. Weitere Informationen zum Styling von Playern finden Sie unter Styling-Spieler.

Kehren Sie zum Konfigurieren der Stileinstellungen zum Players-Modul zurück und führen Sie die folgenden Schritte aus.

  1. Klicken Sie auf den Link für die Beispiel-Player um die Player-Eigenschaftenseite zu öffnen.
  2. Klicken Styling im linken Navigationsmenü.
  3. Wählen Sie a Tastenform spielen und Position der Wiedergabetaste.
  4. In dem Farbthema Klicken Sie im Abschnitt auf Wiedergabetaste Farbwähler.
  5. Klicken Sie auf die rote Farbe auf der Fahrseite des Steuerelements. Klicken Sie in das Quadrat, um einen Rotton auszuwählen.
  6. Klicken Sie außerhalb des Steuerelements, um die Farbauswahl zu reduzieren.
  7. Beachten Sie, dass unter dem Vorschau-Player eine Meldung angezeigt wird, dass Änderungen am Player vorgenommen wurden.

Konfigurieren Sie Social-Sharing-Eigenschaften

Wenn die soziale Freigabe aktiviert ist, wird unten im Player neben dem Abspielkopf ein Freigabesymbol angezeigt. Auf diese Weise können Zuschauer Ihre Inhalte auf beliebten Social-Media-Websites veröffentlichen. Weitere Informationen zum Konfigurieren der sozialen Eigenschaften für einen Spieler finden Sie unter Konfigurieren der sozialen Eigenschaften für einen Spieler.

Führen Sie die folgenden Schritte aus, um die sozialen Einstellungen zu konfigurieren.

  1. Klicken Sozial im linken Navigationsmenü.
  2. Überprüf den Freigabe aktivieren Kontrollkästchen.
  3. Wählen Sie alle Social Media-Websites aus.

Vorschau des Players

Der Vorschau-Player auf der Seite sollte das Styling und die sozialen Änderungen widerspiegeln, die am Player vorgenommen wurden.

Hinweis: Wenn Sie zu dem veröffentlichten Player zurückkehren, den wir zuvor geöffnet haben, und die Seite aktualisieren, werden Sie feststellen, dass diese Änderungen im veröffentlichten Player nicht aktiv sind. Die Farb- und sozialen Änderungen werden erst an den veröffentlichten Player weitergeleitet, wenn die Änderungen im Player-Modul veröffentlicht wurden.

Es ist auch möglich, den Veröffentlichungscode für den Preview-Player zu generieren. Beispielsweise muss die Marketingabteilung möglicherweise alle am Player vorgenommenen Änderungen genehmigen. Sie können ihnen eine URL an den Vorschau-Player senden, damit sie die Änderungen genehmigen können. Führen Sie die folgenden Schritte aus, um den Veröffentlichungscode für den Vorschau-Player zu generieren.

  1. Klicken Sie auf der Startseite des Players-Moduls auf Code & URL einbetten und klicken Sie auf Vorschau des Players. Das Dialogfeld "Preview Player Embed Code" wird angezeigt.
  2. Drücke den Spieler-URL um den Vorschau-Player in einem neuen Browser-Tab zu öffnen.
  3. Der Vorschau-Player sollte die gesamte Breite der Browserseite einnehmen und ein Brightcove-Logo-Overlay aufweisen, das angibt, dass es sich um den Vorschau-Player handelt. Stellen Sie sicher, dass die Änderungen der Farbe und der viralen Werbung sichtbar sind.
  4. Lassen Sie den Preview-Player geöffnet und kehren Sie zum Players-Modul zurück.

Veröffentlichen Sie den endgültigen Spieler

Die Änderungen im Preview-Player sind im veröffentlichten Player erst sichtbar, wenn die Änderungen veröffentlicht wurden. Führen Sie die folgenden Schritte aus, um die Änderungen zu veröffentlichen.

  1. Klicken Sie im Players-Modul auf Veröffentlichen & Einbetten und dann Änderungen veröffentlichen um den Player zu veröffentlichen.
  2. Bestätigen Sie, dass die Veröffentlichung erfolgreich war.
  3. Kehren Sie mit dem veröffentlichten Player zur Browser-Registerkarte zurück. Denken Sie daran, dass der Preview-Player eine Brightcove-Logo-Überlagerung anzeigt.
  4. Aktualisieren Sie die Seite und stellen Sie sicher, dass die Einstellungen für Farbänderung und virale Werbung jetzt im veröffentlichten Player sichtbar sind. Beachten Sie, dass es zu einer leichten Verzögerung kommen kann, bevor die Änderungen im veröffentlichten Player angezeigt werden.
