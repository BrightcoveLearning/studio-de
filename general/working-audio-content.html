Audiodateien aufnehmen

Audioquelldateien können im MP3-, M4A- oder WAV-Format vorliegen und mit dem Upload-Modul hochgeladen werden. Audiodateien können Ihrem Video Cloud-Konto mithilfe von hinzugefügt werden Modul hochladen. Ihre Quelldatei kann im MP3-, M4A- oder WAV-Format vorliegen.

Erstellen eines Aufnahmeprofils

Sie sollten ein benutzerdefiniertes Aufnahmeprofil für Nur-Audio-Inhalte erstellen, das nur Audio-Wiedergaben enthält. Informationen zum Erstellen benutzerdefinierter Aufnahmeprofile finden Sie unter Erstellen von Aufnahmeprofilen für die dynamische Zustellung. Audio-Assets können wie erläutert in einem Brightcove-Player wie ein Video veröffentlicht und abgespielt werden unten.

Hochladen von Poster- und Miniaturbildern

Poster- und / oder Miniaturbilder können Audiodateien hinzugefügt werden. Das Posterbild wird während der Wiedergabe im Brightcove-Player angezeigt, und das Miniaturbild wird in allen Wiedergabelisten angezeigt, zu denen das Element hinzugefügt wurde. Beachten Sie, dass auf iOS-Geräten der native iOS-Player verwendet wird, sodass das Posterbild nicht angezeigt wird.

Informationen zum Hochladen von Poster- und Miniaturbildern finden Sie unter Hochladen von Poster- und Miniaturbildern.

Audiodateien veröffentlichen

Audiodateien werden in jedem Brightcove Player abgespielt. Während der Wiedergabe ist der Videobildschirm entweder leer oder zeigt das Posterbild an, wenn Sie dies tun fügte einen hinzu.

Wenn Sie einen minimaleren Audio-Player ohne Videobildschirm bevorzugen, können Sie zu diesem Zweck einen Player erstellen. Konfigurieren Sie nach dem Erstellen des Players die Größe folgendermaßen:

  • Verwenden Sie eine feste Größe
  • Verwenden Sie ein benutzerdefiniertes Seitenverhältnis
  • Stellen Sie die Höhe auf 80px ein (die Breite kann beliebig sein)
