Berichtsdaten exportieren

Führen Sie die folgenden Schritte aus, um Daten aus einem Bericht zu exportieren.

  1. Navigieren Sie zum Analytics-Modul.
  2. Verwenden Sie die Datumsauswahl, um eine Zeitspanne auszuwählen.
  3. Wählen Sie einen Bericht aus.
  4. Klicken Sie auf den entsprechenden Link, um die Daten als zu exportieren CSV oder XLSX Datei.
  5. Speichern Sie die Datendatei.
