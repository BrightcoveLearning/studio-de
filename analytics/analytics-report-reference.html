Das Video Cloud Analytics-Modul bietet die folgenden Berichte:

  • Leistung
  • Engagement
  • Gerade angesagt
  • Benutzerdefiniert

Eine Zusammenfassung jedes dieser Berichte wird in den folgenden Abschnitten beschrieben.

Leistungsbericht

Der Leistungsbericht zeigt die Leistung für die Videos in Ihrem Konto an, einschließlich Videoansichten, Videoimpressionen, Wiedergaberate, angezeigten Minuten und Engagement-Punktzahl. Mit dem Leistungsbericht können die Daten in Dimensionen unterteilt werden:

  • Video - Zeigt Leistungsmetriken für Videos an
  • Spieler - Zeigt Leistungsmetriken für Spieler an
  • Verkehrsquelle - Zeigt Leistungsmetriken basierend auf der Verkehrsquelle an
  • Domain anzeigen - Zeigt Leistungsmetriken an, die auf der Domäne basieren, in der sich der Betrachter befand, als ein Video angesehen wurde. Befindet sich der Player in einem Iframe, wird die Anzeigedomäne als URL der Seite im Iframe aufgezeichnet. (Dies entspricht dem destination_domain Abmessungen der Analytics-API.)
  • Land - Zeigt Leistungsmetriken an, die auf dem Land basieren, in dem das Video angesehen wurde
  • Gerät - Zeigt Leistungsmetriken für jede Kombination aus Gerätetyp und Gerätebetriebssystem an, die zum Anzeigen eines Videos verwendet wurde
  • Gerätetyp - Zeigt Leistungsmetriken nach Gerätetyp an
  • Gerätebetriebssystem - Zeigt Leistungsmetriken nach Gerätebetriebssystem an
  • Gerätehersteller - Zeigt Leistungsmetriken nach Gerätehersteller an
  • Browsertyp - Zeigt Leistungsmetriken nach Browsertyp an
  • Tag - Zeigt Leistungsmetriken basierend auf dem Tag an

Ausführliche Informationen zum Leistungsbericht finden Sie im Referenz zum Leistungsbericht.

Engagement-Bericht

Der Engagement-Bericht zeigt das Engagement für die Videos in Ihrem Konto. Dies beinhaltet eine Engagement-Punktzahl und die Anzahl der Aufrufe an den Punkten 1%, 25%, 50%, 75% und 100% des Videos. Mit dem Engagement-Bericht können die Daten in Dimensionen unterteilt werden:

  • Video - Zeigt Engagement-Metriken für Videos an
  • Spieler - Zeigt Engagement-Metriken für Spieler an
  • Tag - Zeigt Engagement-Metriken nach Tag an

Ausführliche Informationen zum Engagement-Bericht finden Sie im Referenz zum Engagement-Bericht.

Trending Now Report

Im Bericht "Jetzt im Trend" wird ein Diagramm angezeigt, in dem die Anzahl der gleichzeitigen Betrachter für die letzte Stunde angegeben ist. Der Trending Now-Bericht kann verwendet werden, um die Daten für VoD, Live oder beide Arten von Videos anzuzeigen.

Ausführliche Informationen zum Trending Now-Bericht finden Sie in der Trending Now-Berichtsreferenz.

Benutzerdefinierte Berichte

Mit dem Analytics-Modul können benutzerdefinierte Ad-hoc-Analyseberichte erstellt werden, die in den Formaten durch Kommas getrennte Werte (.csv) oder Microsoft Excel (.xlsx) heruntergeladen werden können.

Hinweis: Benutzerdefinierte Berichte stehen Enterprise- und Pro-Publishern zur Verfügung.

Informationen zum Erstellen benutzerdefinierter Berichte finden Sie unter Erstellen von benutzerdefinierten Analyseberichten im Analysemodul.

Berichtsdaten exportieren

Berichtsdaten können sowohl als durch Kommas getrennte Werte als auch als Excel-Formate heruntergeladen werden.
