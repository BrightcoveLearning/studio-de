Die Video Cloud Analytics-Plattform ist eine hochmoderne Echtzeit-Analyseplattform, die von Brightcove unter Verwendung der neuesten Innovationen in der Big Data-Technologie von Grund auf neu entwickelt und für eine massive Skalierung konzipiert wurde. Die Analyseplattform bietet schnellen und nahezu Echtzeitzugriff auf Analysedaten und enthält eine leistungsstarke offene API, mit der Entwickler zusätzlich zu den in Video Cloud Studio enthaltenen benutzerdefinierten Analyselösungen und Berichten erstellen können.

 

Unabhängig von Ihren spezifischen Zielen für Online-Videos ist das Verständnis Ihres Publikums ein Schlüssel zur Entwicklung einer erfolgreichen Videostrategie. Je besser Sie Ihre Betrachter verstehen, desto einfacher ist es, Inhalte zu erstellen, die sie in Formaten einbinden, die sie problemlos verwenden können. Das Analytics-Modul bietet verschiedene Berichte, mit denen Sie Analysen zu Ihren Videos und Playern abrufen können:

  • Leistungsbericht - Enthält Details zur Anzahl der Videoansichten, Verkehrsquellen, Gerätetypen und Gerätebetriebssysteme
  • Engagement-Bericht - Enthält Details dazu, wie Benutzer mit Ihren Videos interagieren. Wie viel Prozent der Videobetrachter sehen sich an?
  • Gerade angesagt - Gibt die Anzahl der gleichzeitigen Zuschauer für die letzte Stunde an
  • Benutzerdefiniert - Bietet die Möglichkeit, benutzerdefinierte Analyseberichte zu erstellen

Mit diesen Berichten können Sie aktuelle Berichte darüber erstellen, wie viele Zuschauer Sie haben, wie viel Inhalt Ihre Zuschauer sehen, wann sie keine Videos mehr sehen und sogar die Verkehrsquellen für Ihre Inhalte.

Dieses Thema enthält die folgenden Abschnitte, in denen die Verwendung des Analytics-Moduls beschrieben wird:

  • Starten des Analytics-Moduls
  • Auswahl einer Zeitspanne
  • Suche nach einem Video
  • Einschränkungen

Beachten Sie, dass alle im Analytics-Modul angezeigten Daten auch über die Analytics-API verfügbar sind. Weitere Informationen finden Sie in der Video Cloud Analytics API Dokumentation.

Starten des Analytics-Moduls

So starten Sie das Analytics-Modul:

  1. Melden Sie sich bei Video Cloud Studio an.
  2. Öffnen Sie das Analytics-Modul, indem Sie auf klicken Analytik im Navigationsheader.

Auswahl einer Zeitspanne

Verwenden Sie die Datumsauswahl, um den Zeitraum für den Bericht anzugeben, Sie können einen festen Datumsbereich oder einen benutzerdefinierten Datumsbereich auswählen. Beachten Sie, dass bei Auswahl eines einzelnen Tages die Diagramme nach Stunden angezeigt werden.

Berichtszeitzone

Die Daten im Analytics-Modul werden in der Berichtszeitzone des Kontos angezeigt. Die Zeitzone wird unterhalb des Kalendersteuerelements angezeigt.

Die Berichtszeitzone wird verwendet, um Tagesgrenzen für Berichtsdaten zu berechnen. Das Ändern der Einstellung für die Berichtszeitzone wirkt sich nur auf zukünftige Daten aus, und Änderungen werden nicht rückwirkend angewendet. Wenn Sie diese Einstellung ändern, sehen Sie möglicherweise einen flachen Punkt oder eine Spitze in Ihren Daten, wenn Sie sich die Tagesgrenzen ansehen, an denen die Änderung angewendet wurde. Aktualisierungen dieser Einstellung werden möglicherweise nicht sofort wirksam, und die Analysedaten beziehen sich möglicherweise weiterhin auf die alte Einstellung, bis das System die Änderungen verarbeitet.

Die Berichtszeitzone kann als Teil der Kontoeinstellungen konfiguriert werden. Informationen zum Ändern der Berichtszeitzone finden Sie unter Kontoeinstellungen verwalten.

Suche nach einem Video

Mit der Suchfunktion können Sie nach einem bestimmten Video suchen.

Die folgenden Videoeigenschaften werden durchsucht:

  • Name
  • Kurze Beschreibung
  • Lange Beschreibung
  • Stichworte
  • Video ID
  • Referenz ID
  • Alle benutzerdefinierten Felder

Eine Liste der Übereinstimmungen wird unter dem Suchfeld angezeigt.

Durch Klicken auf einen Videolink wird der Bericht "im Kontext" geöffnet. Wenn Sie beispielsweise das Engagement für Ihr Konto anzeigen und dann mithilfe der Suche ein bestimmtes Video suchen, wird durch Klicken auf den Link zum Video der Engagement-Bericht geöffnet. Wenn Sie sich den Leistungsbericht ansehen, wird durch Suchen und Klicken auf einen Link der Leistungsbericht geöffnet.

Eine vollständige Liste der Optionen, die bei der Suche nach Videos im Analytics-Modul verfügbar sind, finden Sie unter So suchen Sie im Analytics-Modul nach Videos.

Hinweis: Die Analytics-Berichte enthalten Analytics für Videos, die aus Ihrem Konto gelöscht wurden. Sie können im Analytics-Modul nicht nach gelöschten Videos suchen. Um einen Bericht über ein gelöschtes Video zu erhalten, müssen Sie die Video-ID für das gelöschte Video kennen und die Analytics-API verwenden, um den Bericht zu generieren.
