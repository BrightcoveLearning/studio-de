Die Playlist-Endbildschirm Plugin ermöglicht die Anzeige einer Wiedergabeliste von Videos auf dem Endbildschirm des Players. Der Betrachter kann dann auf ein Video-Miniaturbild klicken, um ein anderes Video anzusehen. In diesem Player wurde ein Playlist-Endscreen konfiguriert.

Informationen zu anderen Optionen für Player-Endbildschirme finden Sie unter Konfigurieren des Player-Endbildschirms.

Hinweis: Informationen zum programmgesteuerten Konfigurieren eines Wiedergabelisten-Endbildschirms finden Sie in der Entwicklerdokumentation.

Playlist-Endscreen-Plugin konfigurieren

Um das Playlist-Endscreen-Plugin mit dem Players-Modul zu implementieren, gehen Sie folgendermaßen vor:

  1. Öffnen Sie das Players-Modul und suchen Sie den Player.
  2. Klicken Sie auf den Link für den Player, um die Player-Eigenschaften zu öffnen.
  3. Klicken Plugins in der linken Navigation.
  4. Klicken Plugin hinzufügen> Brightcove-Plug-in.
  5. Für die Brightcove-Plug-in , auswählen Playlist-Endbildschirm.
  6. Die Optionen (JSON) enthält die
    • PlaylistId - ID der Playlist, die auf dem Endscreen angezeigt werden soll
    • Titel (optional) - Titel zur Anzeige über der Liste der Videos
    7. {
  "playlistId": "1234567001",
  "title": "Related Videos"
}
  7. Klicken speichern und dann Veröffentlichen der Spieler. Beachten Sie, dass es bis zu 5 Minuten dauern kann, bis die Änderungen in veröffentlichten Playern angezeigt werden.
