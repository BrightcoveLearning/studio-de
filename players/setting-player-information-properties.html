Die Player-Informationseigenschaften können verwendet werden, um den Spielernamen, die Größe, das Seitenverhältnis, die Abmessungen, den Aktualisierungsmodus und die Player-Version festzulegen.

Viele der Player-Eigenschaften können auch dynamisch festgelegt / geändert werden, indem der Player-Veröffentlichungscode bearbeitet wird. Weitere Informationen finden Sie in diesen zugehörigen Dokumenten, je nachdem, ob Sie das verwenden Standard (Iframe) oder Erweitert (In-Page) Code einbetten.

So konfigurieren Sie die Spielerinformationen Eigenschaften, folgen Sie diesen Schritten:

  1. Klicken Spieler im Navigationsheader, um das Players-Modul zu öffnen.
  2. Klicken Sie auf den Link für den Player, den Sie konfigurieren möchten.
  3. Klicken Spielerinformationen im linken Navigationsmenü.
Hinweis: Alle an einem Spieler vorgenommenen Änderungen können bis zu 5 Minuten dauern, bis sie bei veröffentlichten Spielern angezeigt werden.

Die folgenden Eigenschaften für Spielerinformationen können konfiguriert werden:

  • Name und Kurzbeschreibung
  • Spielertyp
  • Dimensionierung
  • Seitenverhältnis
  • Spielerabmessungen
  • Aktualisierungsmodus
  • Player-Version
  • Lokalisierung

Klicken Sie nach dem Vornehmen von Änderungen auf speichern und veröffentlichen Sie dann den Player.

Spielertyp

Das Spielertyp Mit dieser Einstellung wird festgelegt, ob der Player ein einzelnes Video anzeigt oder ob Wiedergabelisten unterstützt werden:

  • Einzelvideo - Ein einzelnes Video wird angezeigt
  • Wiedergabeliste - Der Player unterstützt Wiedergabelisten

Auswählen Wiedergabeliste wird einige zusätzliche aktivieren Styling und Wiedergabe Eigenschaften.

Dimensionierung

Die Größe kann auf eingestellt werden Reaktionsschnell oder Fest.

  • Reaktionsschnell - Die Playergröße wird automatisch basierend auf der Browsergröße angepasst
  • Fest - Die Playergröße basiert auf den Parametern Breite und Höhe
Hinweis: Wenn Sie die auswählen Reaktionsschnell Bei der Größenoption funktioniert der Standardcode (Iframe) gut und der Player und die Wiedergabeliste reagieren. Wenn Sie den Advanced (in-Page-Einbettung) -Code verwenden, reagiert der Player, die Wiedergabeliste jedoch nicht.

Seitenverhältnis

Das Seitenverhältnis Legt das Seitenverhältnis des Players fest.

  • 1:1, 3:2, 4:3, 16:9, 21:9 oder 9:16 - Geben Sie eine Spielerbreite ein und die Höhe wird automatisch bestimmt
  • Benutzerdefiniert - Geben Sie Werte für die Breite und Höhe des Players ein

Spielerabmessungen

Das Spielerabmessungen Steuern Sie die Breite und Höhe des Players.

Update-Modus und Player-Version

Das Aktualisierungsmodus steuert, wie Player-Updates behandelt werden. Das Player-Version wird verwendet, um eine Player-Version auszuwählen. Das Veröffentlichte Player-Version Feld zeigt die aktuelle Player-Version an. Weitere Informationen zu diesen Einstellungen und deren Verwendung finden Sie unter Einstellen des Player-Aktualisierungsmodus und der Version.

Lokalisierung

Das Lokalisierung Das Feld kann verwendet werden, um die Sprache (n) einzuschließen, die Sie für die Spielerlokalisierung verwenden möchten. Weitere Informationen zur Lokalisierung von Spielern finden Sie unter Konfigurieren der Player-Lokalisierung.
