Spieler können so konfiguriert werden, dass ein Endbildschirm angezeigt wird, wenn ein Video endet. Der Endbildschirm kann so konfiguriert werden, dass die Wiedergabe- und Freigabesymbole angezeigt werden, oder er kann so angepasst werden, dass er HTML enthält. Ein benutzerdefinierter Endbildschirm wird unten angezeigt.

Es werden Symbole angezeigt, mit denen der Betrachter das Video wiedergeben oder für beliebte Social-Media-Websites freigeben kann.

Hinweis: Wenn ein Player-Endbildschirm konfiguriert ist, wird er auf einem iOS-Gerät nicht angezeigt, wenn sich der Player im nativen Vollbildmodus befindet.

Gehen Sie folgendermaßen vor, um einen Player-Endbildschirm zu konfigurieren:

  1. Klicken Spieler im Navigationsheader, um das Players-Modul zu öffnen.
  2. Klicken Sie auf den Link für den Player, den Sie konfigurieren möchten.
  3. Klicken Endbildschirme im linken Navigationsmenü.
  4. Wähle aus Player-Endbildschirm Möglichkeit. Folgende Optionen stehen zur Verfügung:
    • Keine - Es wird kein Endbildschirm angezeigt
    • Sozial - Zeigen Sie den Social-Sharing-Dialog an, wenn das Video endet
    • Benutzerdefiniert - Zeigen Sie einen benutzerdefinierten Endbildschirm an, wenn das Video endet
Hinweis: Alle an einem Spieler vorgenommenen Änderungen können bis zu 5 Minuten dauern, bis sie bei veröffentlichten Spielern angezeigt werden.

Sozialer Endbildschirm

Wenn der Sozial Wenn diese Option ausgewählt ist, wird der Dialog zum Teilen von Inhalten angezeigt, wenn das Video endet.

Social Sharing für den Spieler muss aktiviert sein, um den Endbildschirm für Social Sharing verwenden zu können. Wenn die Sozial Wenn die Option ausgewählt ist, wird die soziale Freigabe automatisch aktiviert. Die angezeigten Social-Sharing-Optionen basieren auf den Einstellungen, die im Rahmen der sozialen Einstellungen konfiguriert wurden. Weitere Informationen zu sozialen Einstellungen finden Sie unter Das Soziale konfigurieren Eigenschaften für einen Spieler.

Benutzerdefinierter Endbildschirm

Wenn der Benutzerdefiniert Wenn die Option ausgewählt ist, kann benutzerdefiniertes HTML verwendet werden, um zusätzlichen Text oder Bilder über den Wiedergabe- und Freigabesymbolen anzuzeigen. Um einen benutzerdefinierten Endbildschirm zu konfigurieren, wählen Sie Benutzerdefiniert und fügen Sie HTML ein, um in der angezeigt zu werden Benutzerdefiniertes HTML Eigentum.

Hinweis: Ein Video kann verwendet werden, indem das <video> html-Element zu benutzerdefiniertem HTML hinzugefügt wird.
Hinweis: Die Freigabe muss aktiviert sein, damit das Freigabesymbol auf dem Endbildschirm angezeigt wird.
Hinweis: Wenn die Option Benutzerdefiniert ausgewählt ist und kein HTML angegeben ist, wird die Wiedergabeschaltfläche angezeigt, wenn das Video endet.
