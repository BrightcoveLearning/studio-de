Auf der Seite zur Verwaltung von Aufnahmeprofilen können neue ältere Aufnahmeprofile erstellt werden. Aufnahmeprofile werden mit JSON definiert. Wenn Sie ein neues Legacy-Profil erstellen, ist es hilfreich, das zu überprüfen Standard-Aufnahmeprofile für die Legacy-Transcodierung dokumentieren und suchen Sie ein Profil, das Ihren Anforderungen entspricht. In Studio können Sie das Profil dann einfach duplizieren und dann mit Ihren Änderungen bearbeiten.

Hinweis: Informationen zum Erstellen von Aufnahmeprofilen, die DRM verwenden, finden Sie in der MPEG-DASH-Video Entwicklerdokument.

Hinweis: Jedes Profil kann maximal 25 Wiedergaben erstellen.  Poster- und Miniaturbilder zählen maximal 25. 

So erstellen Sie ein neues Legacy-Aufnahmeprofil:

  1. Drücke den ADMINISTRATOR Link im Studio-Header.
  2. Drücke den Profile aufnehmen Verknüpfung.
  3. Suchen Sie ein Legacy-Profil zum Duplizieren. Klicken Sie auf den Profilnamen, um den Profileditor zu öffnen.
  4. Klicken Profil duplizieren. Der Profileditor wird mit dem neuen Profil geöffnet.
  5. Scrollen Sie zum Anfang des Profils. Bearbeiten Sie die name Element, um dem Profil einen aussagekräftigen Namen zu geben. Dieser Name wird in Video Cloud Studio angezeigt.
  6. Bearbeiten Sie die description Element, um dem Profil eine aussagekräftige Beschreibung zu geben.
  7. Bearbeiten Sie die Wiedergabeinformationen nach Bedarf. Die Standardgröße für die Poster- und Miniaturbilder befindet sich am Ende des Profils.
  8. Klicken speichern.

Hinzufügen eines Wasserzeichens zu einem Profil

Zu jeder Wiedergabe kann ein Wasserzeichen hinzugefügt werden, indem Sie das hinzufügen watermarks Element zu jedem rendition::

       Wasserzeichen": [{ "y": -20, "x": -10, "width": "20%", "url": "http://www.mysite.com/companylogo.png" }]

Das watermarks Element hat die folgenden Parameter.

Parameter Typ Beschreibung
url Schnur URL für das Wasserzeichenbild
width Schnur Breite in Pixel oder Prozent der Rahmenbreite; z. B. 20 oder 10%
height Schnur Höhe in Pixel oder Prozent der Rahmenbreite; z. B. 20 oder 10%
x Schnur Abstand von der linken Kante zur Bildmitte als Pixel oder Prozent der Bildbreite; z. B. 20 oder 10% (verwenden Sie negative Werte wie -10, um das Wasserzeichen relativ zur rechten Kante zu positionieren)
y Schnur Abstand von Oberkante zu Mitte wenn Bild als Pixel oder Prozent der Bild-Höhe; z. B. 20 oder 10% (verwenden Sie negative Werte wie -10, um das Wasserzeichen relativ zur Unterkante zu positionieren)
