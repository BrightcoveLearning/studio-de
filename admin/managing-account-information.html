Die Seite Kontoinformationen verwaltet den Kontonamen und die Postanschrift und kann zum Klonen von Konten verwendet werden. Beachten Sie, dass Sie ein Benutzer mit der Administratorrolle sein müssen, um auf die Seite "Kontoverwaltung" zugreifen zu können.

So greifen Sie auf die Seite "Kontoverwaltung" zu:

  1. Melden Sie sich bei Studio an.
  2. Klicken Sie in der Navigationsüberschrift auf ADMINISTRATOR und dann Kontoinformationen.

Informationen zum Konto werden angezeigt.

Um Ihre Kontoinformationen zu bearbeiten, klicken Sie auf Kontoinformationen bearbeiten.

Bearbeiten Sie die entsprechenden Werte und klicken Sie dann auf speichern. Beachten Sie, dass die Konto-ID nicht geändert werden kann.

Klonen von Konten

Wenn Ihr Konto für das Klonen von Konten aktiviert ist, a Neuen Account erstellen Die Schaltfläche wird angezeigt. Mit der Funktion zum Klonen von Konten können neue Video Cloud-Konten erstellt werden. Wenn ein neues Konto erstellt wird, wird der Brightcove-Support benachrichtigt und arbeitet dann mit Ihrem Account Manager zusammen, um das neue Konto zu überprüfen (basierend auf Ihrem Vertrag). Nach der Überprüfung wird das Konto mit den in Ihrem Vertrag beschriebenen Funktionen konfiguriert. Wenn das Konto vollständig konfiguriert ist, werden Sie benachrichtigt. Während der Konfiguration können dem Konto neue Benutzer hinzugefügt werden.

Hinweis: Das Klonen von Konten steht nur Kontoadministratoren mit Enterprise-Konten zur Verfügung.

Um das Konto zu klonen, klicken Sie auf Neues Konto erstellen, geben Sie einen Kontonamen ein und klicken Sie dann auf Speichern.
