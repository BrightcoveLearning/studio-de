Aufnahmeprofile enthalten die Transcode-Einstellungen, die beim Transcodieren von Videos verwendet werden. Der angezeigte Aufnahmeprofil-Editor ändert sich basierend auf den für das Konto konfigurierten Transcodierungsoptionen.

Anmerkungen:
  • Die Möglichkeit zum Erstellen und Bearbeiten von Aufnahmeprofilen steht nur Publishern mit Enterprise- und Professional-Konten zur Verfügung.
  • Wenn Sie in der Lage sein möchten, mit einem benutzerdefinierten Profil aufgenommene Videos erneut zu codieren, müssen Sie muss den digitalen Master archivieren (wie alle Standardprofile). Sehen Archivierungsmeister Einzelheiten dazu finden Sie hier.

Klicken Sie auf die Verwaltungsseite Ingest Profiles ADMINISTRATOR Link und dann Profile aufnehmen. Eine Liste der Aufnahmeprofile wird angezeigt.

Hinweis: Um auf die Seite „Profile aufnehmen“ zugreifen zu können, müssen Sie ein Video Cloud Studio-Benutzer mit der Administratorrolle sein.

Ihr Video Cloud-Konto wird mit einem oder mehreren Aufnahmeprofilen konfiguriert. Alle Video Cloud-Konten sind mit einem Standardsatz von Aufnahmeprofilen konfiguriert, die nicht bearbeitet oder gelöscht werden können. Diese sind mit einem gekennzeichnet Art von Standard VOD im Studio. Das Profilversion Zeigt die Art der Transcodierung an, die mit dem Profil verwendet wird. Spezifische Details zu den Wiedergaben, die mit den von Brightcove gelieferten Aufnahmeprofilen bereitgestellt werden, finden Sie unter Standard-Aufnahmeprofile für die dynamische Bereitstellung und die kontextbezogene Codierung.

Erstellen eines neuen dynamischen Übermittlungsprofils

Auf der Verwaltungsseite "Aufnahmeprofile" können neue Aufnahmeprofile erstellt werden. Aufnahmeprofile werden mit JSON definiert.

So erstellen Sie ein neues dynamisches Übermittlungs-Aufnahmeprofil:

  1. Drücke den ADMINISTRATOR Link im Studio-Header.
  2. Drücke den Profile aufnehmen Verknüpfung.
  3. Klicken Neues Profil> Dynamische Zustellung. Der Profileditor wird geöffnet.
  4. Geben Sie ein Name und Beschreibung für das Profil.
  5. Entscheide ob Stammdateien speichern. Durch das Speichern des Masters bleibt eine Kopie der Masterdatei erhalten, und das Video kann in Zukunft erneut codiert werden, ohne dass es erneut hochgeladen werden muss. Weitere Informationen zum Archivieren von Mastern finden Sie in der Archivierungsmeister Dokument.
  6. Geben Sie die Breite und Höhe für die Poster und Miniaturansicht Bilder.
  7. Wählen Sie eine Einstellung für die Bilder:
    • Nehmen Sie Bilder beim Hochladen automatisch auf - Video Cloud erfasst Poster- und Miniaturbilder, wenn das Video hochgeladen wird. Bilder werden am 50% -Punkt im Video aufgenommen.
    • Nehmen Sie keine Bilder auf. Ich werde meine eigenen Bilder hochladen oder aufnehmen - Poster- und Miniaturbilder werden angezeigt gefangen oder hochgeladen Verwenden des Medienmoduls.
  8. Fügen Sie die Wiedergaben hinzu, die Sie für das Profil benötigen. Stellen Sie sicher, dass Sie sowohl Video- als auch Audio-Wiedergaben auswählen, um sicherzustellen, dass Ihre Videos mit Audio abgespielt werden. Fügen Sie auch progressive (MP4) Wiedergaben hinzu, wenn für das Video herunterladbare Wiedergaben verfügbar sein sollen. Klicken Rendition hinzufügen.
    Anmerkungen:
    • Klicken Sie auf, um die verfügbaren Audiodaten und Wiedergaben anzuzeigen Renditionsliste anzeigen Verknüpfung.
    • Wenn Sie eine 1080p-Wiedergabe zum Download bereitstellen möchten, müssen Sie dies tun Erstellen Sie ein benutzerdefiniertes Aufnahmeprofil und stellen Sie sicher, dass mindestens eine der folgenden Wiedergaben enthalten ist:
      • progressive3500
      • progressive4000
    • Während ist es möglich, ein Profil zu erstellen, das generiert wird nur Progressive Wiedergaben: Wenn Sie Videos mit diesem Profil hochladen, werden Miniatur- und Posterbilder angezeigt nicht während der Transcodierung erfasst werden.
  9. Klicken Sie weiter Rendition hinzufügen um zusätzliche Wiedergaben hinzuzufügen.
  10. Verwenden Sie die Dropdown-Listen Codec und Wiedergabeversion, um eine Audio-, Video- (H264, HEVC )- oder Progressive (MP4) -Formatvariante auszuwählen, die aufgenommen werden soll. Die Wiedergabedetails werden in der Tabelle angezeigt.
  11. Klicken speichern um das neue Profil zu speichern.

Erstellen von benutzerdefinierten Profilen für Social Clipping

Benutzerdefinierte Aufnahmeprofile können auch für Social Clipping verwendet werden. Im Allgemeinen sind die in den Brightcove-Standard-Aufnahmeprofilen enthaltenen Wiedergaben mit geringerer Qualität für Anwendungsfälle der sozialen Verteilung nicht erforderlich. Daher können benutzerdefinierte Schnitt-Aufnahmeprofile in der Regel auf eine kleine Anzahl von Wiedergaben mittlerer bis hoher Qualität beschränkt werden. Beispielsweise ist ein benutzerdefiniertes Clipping-Aufnahmeprofil, das die Videowiedergaben video3500, video2000 und video1700 enthält, für die meisten Anwendungsfälle der sozialen Verteilung mehr als ausreichend. Durch die Verwendung eines benutzerdefinierten Profils für Social Clipping können Sie zusätzliche Transformationen auf Clips anwenden, z. B. das Hinzufügen von Wasserzeichenbildern. Um ein benutzerdefiniertes Profil mit einem Wasserzeichen zu erstellen, wenden Sie sich an Brightcove-Unterstützung.

Duplizieren von Aufnahmeprofilen

Wenn Sie ein Profil erstellen müssen, das einem vorhandenen Profil ähnelt, können Sie ein vorhandenes Profil duplizieren und dann bearbeiten, um die erforderlichen Änderungen vorzunehmen.

So duplizieren Sie ein Aufnahmeprofil:

  1. Drücke den ADMINISTRATOR Link im Studio-Header.
  2. Drücke den Profile aufnehmen Verknüpfung.
  3. Klicken Sie auf das Aufnahmeprofil, um es zu duplizieren.
  4. Klicken Duplikat.
  5. Verwenden Sie den Profileditor, um den Profilnamen und die Definition zu bearbeiten. Wiedergaben können entfernt werden, indem Sie auf das Symbol zum Entfernen klicken () neben einer Wiedergabe.
  6. Klicken speichern.

DRM-Unterstützung in Profilen

Wenden Sie sich an den Brightcove-Support, um die Standardeinstellungen für die DRM-Verpackung Ihrer Videos festzulegen. Sie müssen in Ihren Aufnahmeprofilen keine DRM-Verpackung angeben. Wenn Ihr Konto für DRM aktiviert ist und Sie einige Titel ohne DRM liefern möchten, bearbeiten Sie die Videoeigenschaften im Medienmodul und setzen Sie das DRM-Flag für das Video auf Kein DRM.

Hinweis: Informationen zum Erstellen eines DRM-fähigen Brightcove Players finden Sie im Thema DRM-Plugin.

Aufnahmeprofile bearbeiten

Auf der Verwaltungsseite "Aufnahmeprofile" können Sie Aufnahmeprofile bearbeiten. Beachten Sie, dass der Standardsatz von Ingest-Profilen (mit Standard VOD gekennzeichnet) nicht bearbeitet werden kann.

So bearbeiten Sie ein Aufnahmeprofil:

  1. Drücke den ADMINISTRATOR Link im Studio-Header.
  2. Drücke den Profile aufnehmen Verknüpfung.
  3. Klicken Sie zum Bearbeiten auf das Aufnahmeprofil.
  4. Verwenden Sie den Profileditor, um die Profildefinition zu bearbeiten. Wiedergaben können entfernt werden, indem Sie auf das Symbol zum Entfernen klicken () neben einer Wiedergabe.
  5. Klicken speichern.
Hinweis: Sie können einem Profil mit dem Basiseditor Standarddarstellungen hinzufügen, aber benutzerdefinierte Wiedergaben muss mit hinzugefügt werden erweiterter Editor.

Aufnahmeprofile löschen

Auf der Seite zur Verwaltung von Aufnahmeprofilen können nicht mehr benötigte Aufnahmeprofile gelöscht werden. Beachten Sie, dass der Standardsatz von Aufgabeprofilen (mit Standard VOD gekennzeichnet) nicht gelöscht werden kann, aber ausgeblendet werden kann, wenn sie nicht benötigt werden.

So löschen Sie ein Aufnahmeprofil:

  1. Drücke den ADMINISTRATOR Link im Studio-Header.
  2. Drücke den Profile aufnehmen Verknüpfung. Die Seite zur Verwaltung von Aufnahmeprofilen wird geöffnet und zeigt die Aufnahmeprofile an, die für Ihr Konto erstellt wurden.
  3. Klicken Sie auf das Löschen-Symbol () neben dem Aufnahmeprofil oder klicken Sie auf den Profilnamen und dann auf das Löschen Taste.
  4. Bestätigen Sie den Löschvorgang durch Klicken auf Profil löschen.

Auswählen eines Standard-Aufnahmeprofils

Ein Standard-Aufnahmeprofil kann für wann konfiguriert werden Videos werden hochgeladen und wann Erstellen von Clips mit dem Modul "Sozial". Das Upload-Standardprofil ist mit . gekennzeichnet neben dem Profilnamen. Das Standardprofil für Social Clipping ist mit . gekennzeichnet neben dem Profilnamen.

So wählen Sie ein Standard-Aufnahmeprofil zum Hochladen / Ausschneiden aus:

  1. Drücke den ADMINISTRATOR Link im Studio-Header.
  2. Drücke den Profile aufnehmen Verknüpfung. Die Verwaltungsseite für Aufnahmeprofile wird geöffnet und zeigt die Aufnahmeprofile an, die für das Konto erstellt wurden.
  3. Klicken Sie auf den Namen des Aufnahmeprofils für das Profil, das Sie als Standard festlegen möchten.
  4. Drücke den Hochladen oder Social Clipping Link oben auf der Seite, um das aktuelle Profil als Standard festzulegen.
Hinweis: Nur Dynamic Delivery-Profile können als Standardprofil für Social Clipping festgelegt werden. CAE-Profile werden nicht unterstützt.

Ein- und Anzeigen von Aufnahmeprofilen

Aufnahmeprofile können ausgeblendet werden, indem Sie auf den mit dem Profil verknüpften Schieberegler klicken Ausblenden Position. Dies ist nützlich, um Standardprofile auszublenden, die möglicherweise nicht benötigt werden.

Versteckte Aufnahmeprofile werden nicht in der Liste Aufnahmeprofil im Upload-Modul angezeigt.

Verwenden des erweiterten Editors

Aufnahmeprofile werden mit JSON definiert. Um die JSON-Syntax für ein Profil anzuzeigen, bearbeiten Sie ein Profil und klicken Sie dann auf Erweiterter Editor Tab. Die JSON-Darstellung des Profils wird angezeigt.

Während der JSON hier bearbeitet werden kann, empfiehlt Brightcove, Profile mit dem einfachen Editor zu bearbeiten, um die Wahrscheinlichkeit von Fehlern zu verringern. Informationen zu den verfügbaren Attributen finden Sie in der API-Feldreferenz für Aufnahmeprofile für die dynamische Bereitstellung und die kontextbezogene Codierung Thema.
