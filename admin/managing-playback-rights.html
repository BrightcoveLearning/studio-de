Wiedergaberechte werden verwendet, um die Videowiedergabe einzuschränken. Nachdem Wiedergaberechte erstellt wurden, können sie mithilfe des Medienmoduls Videos zugewiesen werden.

Klicken Sie auf, um auf die Verwaltungsseite für Wiedergaberechte zuzugreifen ADMINISTRATOR Link und dann Wiedergaberechte. Eine Liste der Wiedergaberechte wird angezeigt.

Hinweis: Um auf die Seite „Wiedergaberechte“ zugreifen zu können, müssen Sie ein Video Cloud Studio-Benutzer mit der Administratorrolle sein.
Wiedergaberechte werden Videos im Internet zugewiesen Wiedergabebeschränkungen Abschnitt der Seite mit den Videoeigenschaften im Medienmodul.

Wiedergaberechte mit a Status von Inaktiv kann im Medienmodul keinen Videos zugewiesen werden.

Wiedergaberechte erstellen

Führen Sie die folgenden Schritte aus, um Wiedergaberechte zu erstellen.

  1. Melden Sie sich als Administrator bei Video Cloud Studio an.
  2. Klicken Sie in der Navigationsüberschrift auf ADMINISTRATOR und dann Wiedergaberechte. Beachten Sie, dass Sie ein Kontoadministrator sein müssen, um auf das ADMIN-Menü zugreifen zu können.
  3. Klicken Neue Wiedergaberechte.
  4. Geben Sie ein Name der Wiedergaberechte.
  5. Überprüfung Wiedergabe ohne Token zulassen ermöglicht die Wiedergabe von Anforderungen ohne JSON Web Token (JWT).
    Hinweis: Das Deaktivieren dieser Option wirkt sich nicht auf die für dieses Wiedergaberecht festgelegten Einschränkungen aus. Für die Wiedergabeanforderung ist jedoch eine gültige JWT erforderlich, die programmgesteuert auf Spielerebene festgelegt wird. Weitere Informationen finden Sie im Entwicklerdokument Verwenden der Wiedergabeautorisierung DRM mit dem Brightcove Player.
  6. Prüfen Standardmäßig verwenden um dies zur Standardwiedergabe zu machen. Dadurch werden die konfigurierten Einschränkungen für Wiedergaberechte auf hochgeladene Videos angewendet.
    Hinweis: Das Standardwiedergaberecht wird nicht zugewiesen, wenn ein Video erneut transcodiert wird.
  7. Klicken speichern.
Hinweis: Die maximale Anzahl von Wiedergaberechten, die Sie in einem Konto haben können, beträgt 100. Wenn Sie das Limit erhöhen müssen, wenden Sie sich bitte an den Brightcove-Support.

Klicken Sie in der linken Navigation auf einen Einschränkungslink, um einer Wiedergabe rechts Einschränkungen hinzuzufügen:

  • Geografisch
  • Domäne
  • Proxy
  • Zeit

Wiedergaberechte bearbeiten

Gehen Sie folgendermaßen vor, um die Wiedergaberechte zu bearbeiten:

  1. Klicken Sie rechts auf eine Wiedergabe.
  2. Nehmen Sie die erforderlichen Änderungen vor und klicken Sie auf speichern.

Duplizieren von Wiedergaberechten

Gehen Sie folgendermaßen vor, um ein Wiedergaberecht zu duplizieren:

  1. Klicken Sie rechts auf eine Wiedergabe.
  2. Klicken Duplikat.
  3. Nehmen Sie die erforderlichen Änderungen vor und klicken Sie auf speichern.

Wiedergaberechte aktivieren / deaktivieren

Wiedergaberechte können nicht gelöscht werden, sie können jedoch deaktiviert werden, sodass sie im Medienmodul keinen Videos zugewiesen werden können. Gehen Sie folgendermaßen vor, um die Wiedergaberechte zu aktivieren / deaktivieren:

  1. Klicken Sie rechts auf eine Wiedergabe.
  2. Klicken aktivieren Sie um eine Wiedergabe richtig zu aktivieren oder Deaktivieren um eine Wiedergabe richtig zu deaktivieren. Deaktivierte Wiedergaberechte können Videos im Medienmodul nicht zugewiesen werden.

Zuweisen von geografischen Einschränkungen

Geografische Einschränkungen ermöglichen es Ihnen, die Videowiedergabe basierend auf Land, Postleitzahl, IPs und ausgewiesenem Marktgebiet einzuschränken. Gehen Sie folgendermaßen vor, um einem Wiedergaberecht geografische Einschränkungen zuzuweisen:

  1. Klicken Sie rechts auf eine Wiedergabe.
  2. Klicken Sie in der linken Navigation auf Geografische Einschränkungen.

Die Globale Regel kann eingestellt werden auf Alles erlauben oder Alles blockieren.

  • Alles erlauben
    • Wird verwendet, wenn Sie eine kleine Anzahl von Gruppen auf die schwarze Liste setzen möchten, die keinen Zugriff haben
    • Wählen Sie diese Option, um die Anzeige für alle zuzulassen, und fügen Sie dann eine Liste mit Ländern, Postleitzahlen, IPs und DMAs auf der schwarzen Liste hinzu, deren Anzeige verhindert ist
  • Alles blockieren
    • Wird verwendet, wenn Sie eine kleine Anzahl von Gruppen auf die Whitelist setzen möchten, die Zugriff haben
    • Wählen Sie diese Option, um die Anzeige für alle zu blockieren, und fügen Sie dann eine Liste mit Ländern, Postleitzahlen, IPs oder DMAs auf der weißen Liste hinzu, die sehen dürfen

Geografische Beschränkungen können hinzugefügt werden, um die Anzeige zuzulassen/zu verhindern, basierend auf:

  • Länder
  • Postleitzahlen
  • IP-Einschränkungen
  • Ausgewiesener Marktbereich (DMA)

Beschränkung des Zugangs zu Ländern

Länderwiedergaberechte können verwendet werden, um die Wiedergabe auf bestimmte Länder zu beschränken.

  1. Wählen Sie die Länder aus, deren Anzeige verhindert (auf der schwarzen Liste) oder erlaubt (auf der weißen Liste) ist. Während der Eingabe wird die Auswahlliste gefiltert.
  2. Klicken speichern.

Beschränkung des Zugangs zu Postleitzahlen

Die Wiedergaberechte für Postleitzahlen können verwendet werden, um die Anzeige auf bestimmte Postleitzahlen zu beschränken.

  1. Geben Sie eine Liste mit Postleitzahlen ein, deren Anzeige verhindert (auf der schwarzen Liste) oder erlaubt (auf der weißen Liste) ist. Den Postleitzahlen sollte beispielsweise ein Großbuchstabe, eine aus zwei Buchstaben bestehende Landesvorwahl und ein Bindestrich vorangestellt werden. US-90045.

Es ist auch möglich, eine .csv-Datei mit Postleitzahlen hochzuladen. Klicken Laden Sie CSV hoch um eine hochzuladende Datei zu durchsuchen und auszuwählen.

Hinweis: Durch das Hochladen einer .csv-Datei wird der Inhalt des Feldes Postleitzahlen-Feldes ersetzt.

Einschränken des Zugriffs auf IPs

IP-Wiedergaberechte können verwendet werden, um die Anzeige auf einen IP-Adressbereich zu beschränken.

  1. Geben Sie einen Bereich von IP-Adressen ein, deren Anzeige verhindert (auf der schwarzen Liste) oder erlaubt (auf der weißen Liste) wird. CIDR-Blocknotation wird ebenfalls unterstützt.
  2. Klicken speichern.

Einschränken des Zugriffs auf DMAs

Ein Designated Market Area (DMA), auch als Medienmarkt bezeichnet, ist eine Region der Vereinigten Staaten, die verwendet wird, um Fernseh- und Radiomärkte zu definieren. Wiedergaberechte können verwendet werden, um die Anzeige basierend auf DMAs einzuschränken.

  1. Geben Sie eine Liste von DMAs ein, deren Anzeige verhindert (auf der schwarzen Liste) oder erlaubt (auf der weißen Liste) ist
  2. Klicken speichern.

Zuweisen von Domäneneinschränkungen

Mit Domäneneinschränkungen können Sie die Videowiedergabe basierend auf der Domäne einschränken. Gehen Sie folgendermaßen vor, um einem Wiedergaberecht Domäneneinschränkungen zuzuweisen:

  1. Klicken Sie rechts auf eine Wiedergabe.
  2. Klicken Sie in der linken Navigation auf Domain-Einschränkungen.
  3. In dem Domains auf der schwarzen Liste Geben Sie in das Feld die Domänen ein, deren Anzeige verhindert werden soll.
  4. In dem Whitelist-Domains Geben Sie im Feld die Domänen ein, die angezeigt werden dürfen.
  5. Klicken speichern.

Zuweisen von Proxy-Einschränkungen

Mit Proxy-Einschränkungen können Sie die Videowiedergabe basierend auf bestimmten Verbindungstypen einschränken. Gehen Sie folgendermaßen vor, um einem Wiedergaberecht Proxy-Einschränkungen zuzuweisen:

Hinweis: Da VPN-Anbieter ihre Dienste weiter verbessern, kann sich die Auflösung von IP-Adressen von dem unterscheiden, was Sie für blockierte IP-Adressen definiert haben. Aus diesem Grund ist die Proxyblockierung nicht zu 100% garantiert.
  1. Klicken Sie rechts auf eine Wiedergabe.
  2. Klicken Sie in der linken Navigation auf Proxy-Einschränkungen.
  3. Wählen Sie die zu blockierenden Proxy-Typen aus.
  4. Klicken speichern.

Zeitbeschränkungen zuweisen

Zeitbeschränkungen ermöglichen es Ihnen, die Videowiedergabe basierend auf einem bestimmten Startdatum zu begrenzen. Um einem Wiedergaberecht Zeitbeschränkungen zuzuweisen, gehen Sie folgendermaßen vor:

  1. Klicken Sie rechts auf eine Wiedergabe.
  2. Klicken Sie in der linken Navigation auf Zeitbeschränkungen.
  3. Klicken Sie auf das Kalendersteuerelement, um a auszuwählen Startdatum.
  4. (Optional) Klicken Enddatum hinzufügen um ein Enddatum auszuwählen.
  5. Klicken speichern.
