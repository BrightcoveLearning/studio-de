Wie viele Profile soll ich haben?

Sie können bis zu 200 benutzerdefinierte Profile definieren. Sie sollten jedoch im Allgemeinen nur so viele erstellen, wie Sie benötigen, um Verwirrung bei der Auswahl von Profilen für Aufnahmeaufträge zu vermeiden.

Wie verwalte ich meine Profile?

Sie können die Seite „Profilverwaltung aufnehmen“ in Studio verwenden, um Ihre Aufnahmeprofile zu verwalten. Innerhalb dieser Schnittstelle stehen verschiedene Optionen zur Verfügung, darunter:

  • Festlegen Ihres Standardprofils
  • Neue Profile erstellen
  • Klonen aus vorhandenen Profilen
  • Benutzerdefinierte Profile bearbeiten
  • Nicht benötigte Profile entfernen

Wenn Sie programmgesteuerten Zugriff auf Ihre Aufnahmeprofile benötigen, können Sie mithilfe von benutzerdefinierte Profile erstellen, lesen, aktualisieren und löschen REST-API für Aufnahmeprofile. Beachten Sie, dass Sie dies einrichten müssen OAuth-Anmeldeinformationen.

Wie viele Wiedergaben kann ein Profil haben?

EIN Dynamische Lieferung Profil kann bis zu 37 Wiedergaben haben. Beachten Sie, dass Sie für benutzerdefinierte Dynamic Delivery-Profile nur die auswählen können vordefinierte Wiedergaben die derzeit verfügbar sind.

Wenn Sie ein benutzerdefiniertes Context Aware Encoding-Profil erstellen, min_renditions hat einen Mindestwert von 1 und max_renditions hat einen Maximalwert von 15.

Beachten Sie, dass unnötige Wiedergaben Ihre verwalteten Speicherkosten erhöhen. Sie sollten daher keine Wiedergaben erstellen, die angesichts der Art der von Ihnen veröffentlichten Videos und der Zielgruppen und Geräte, auf die Sie abzielen, wahrscheinlich nicht abgespielt werden.

Die Standardprofile in Ihrem Konto bieten ein gutes Seherlebnis über typische Viewer-Bandbreiten und die am häufigsten verwendeten Geräte.

Benutzerdefinierte Profile benennen

Benutzerdefinierte Profilnamen sollten keine Leerzeichen oder Sonderzeichen enthalten. Verwenden Sie Buchstaben und Zahlen und ersetzen Sie Leerzeichen durch Bindestriche (-) oder Unterstriche (_).

Um Benutzerfehler zu vermeiden, versuchen Sie, Profile so zu benennen, dass den Personen, die Videos hochladen, klar mitgeteilt wird, wofür das Profil verwendet werden soll.

Welches Profil soll ich als Standard festlegen?

Jedes Standard- oder benutzerdefinierte Profil kann entweder in Studio oder über die Ingest Profile-API als Standard festgelegt werden. Die Standardeinstellung wird nur verwendet, wenn der Benutzer kein bestimmtes Profil angibt (entweder im Upload-Modul oder mithilfe der Dynamic Ingest-API).

Wir empfehlen, dass Sie das Standardprofil auf ein Profil festlegen, das keine sehr schlechte Wahl wäre, wenn der Benutzer vergisst, ein bestimmtes Profil anzugeben.
