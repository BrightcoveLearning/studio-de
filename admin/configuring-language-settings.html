In den Spracheinstellungen definieren Sie eine Primärsprache und fügen dann zusätzliche Metadatensprachen hinzu und verwalten diese. Die Primärsprache ist die Sprache, die Ihr Konto derzeit für Videometadaten verwendet. Bevor Sie Ihrem Konto weitere Sprachen hinzufügen, werden Ihre vorhandenen Metadaten der Sprache zugeordnet, die Sie als Primärsprache definieren. Die Primärsprache wird anfänglich angezeigt, wenn Sie die Videoeigenschaften im Medienmodul bearbeiten.

Festlegen einer Primärsprache

So stellen Sie die Primärsprache ein:

  1. Melden Sie sich bei Video Cloud Studio an.
  2. Klicken Sie in der Navigationsüberschrift auf ADMINISTRATOR und dann Spracheinstellungen. Beachten Sie, dass Sie ein Kontoadministrator sein müssen, um auf das ADMIN-Menü zugreifen zu können.
  3. Geben Sie Ihre Hauptsprache ein. Während der Eingabe wird eine Liste der übereinstimmenden Sprachen angezeigt.
  4. Klicken Sie auf eine Sprache, um sie auszuwählen.
  5. Klicken Sprache speichern.
Um die Primärsprache zu ändern, klicken Sie auf die Sprache, wählen Sie eine neue Sprache aus und klicken Sie dann auf speichern.

Hinzufügen weiterer Sprachen

Nach dem Speichern einer Primärsprache können zusätzliche Metadatensprachen hinzugefügt werden. Durch Hinzufügen zusätzlicher Sprachen können Sie zusätzliche mehrsprachige Metadaten in das Medienmodul eingeben. Führen Sie die folgenden Schritte aus, um weitere Sprachen hinzuzufügen.

  1. Klicken Sprache hinzufügen.
  2. Geben Sie Ihre Sprache ein. Während der Eingabe wird eine Liste der übereinstimmenden Sprachen angezeigt.
  3. Klicken Sie auf eine Sprache, um sie auszuwählen.
  4. Klicken Sprache speichern. Die Sprache wird der Tabelle Zusätzliche Sprachen hinzugefügt.
  5. Klicken Sprache hinzufügen um nach Bedarf weitere Sprachen hinzuzufügen.

Um den Sprachnamen zu bearbeiten (wie er im Medienmodul angezeigt wird), klicken Sie darauf, nehmen Sie die gewünschten Änderungen vor und klicken Sie dann auf speichern. Um eine Sprache zu löschen, klicken Sie auf das Löschsymbol ().

Hinweis: Wenn Sie eine verwendete Sprache löschen, werden alle mit der Sprache verknüpften Metadaten ausgeblendet. Um die zugehörigen Metadaten wiederherzustellen, fügen Sie die Sprache erneut hinzu.
