Mit der Option Viewer-Daten verwalten können Sie nach einem Benutzer suchen und diesen entweder entfernen oder die Daten dieses Benutzers exportieren.

Anmerkungen:
  • Normalerweise wird bei der Integration mit einer Marketing-Automatisierungsplattform (MAP-Integration) die Einstellung der Datenaufbewahrungsdauer verwendet, um zu steuern, wie lange Daten aufbewahrt werden
  • Sie müssen ein Kontoadministrator sein, um auf das Menü "Viewer-Daten verwalten" zugreifen zu können

Führen Sie die folgenden Schritte aus, um die Viewer-Daten zu verwalten.

  1. Öffnen Sie das Modul Kampagne.
  2. Klicken Viewer-Daten verwalten in der linken Navigation.
  3. Geben Sie den Benutzer ein, nach dem gesucht werden soll. In der Regel ist dies einer der folgenden Werte:
    • Email - Bei Verwendung von Kampagnen-Tracking und Verwendung eines Kampagnen-Lead-Formulars oder mit SSO-Integrationen
    • Benutzeridentifikation - Bei Verwendung von SSO oder benutzerdefinierten Integrationen
    • Benutzerdefinierte GUID - Für anonyme Benutzer
    • Externe GUID - Wenn eine MAP-Integration verwendet wird
  4. Wenn eine Übereinstimmung gefunden wird, wählen Sie eine Aktion aus:
    • Viewer-Daten löschen - Entfernen Sie Anzeige- und Lead-Daten für den Benutzer
    • Viewer-Daten exportieren - Exportieren Sie den Lead und zeigen Sie Ereignisdaten für den Benutzer an. Die Lead-Daten enthalten die Daten, die in ein Campaign-Lead-Formular eingegeben wurden. Die Ansichtsereignisdaten enthalten alle Ansichtsereignisse für den Viewer.
