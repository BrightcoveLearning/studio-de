Das Campaign-Modul bietet die Möglichkeit, ein Video Cloud-Video zu Eloqua-Landingpages als eine Art von Cloud-Inhalt hinzuzufügen. Das Video wird einer Landing Page im WYSIWYG-Editor der Eloqua Landing Page hinzugefügt.

Um einer Eloqua-Landingpage ein Video hinzuzufügen, müssen die folgenden Schritte ausgeführt werden.

  1. Erstellen Sie eine Landingpage in Eloqua
  2. Wählen Sie ein Video und einen Player aus

Erstellen einer Eloqua-Landingpage

Eloqua-Zielseiten können als Ziel für Besucher verwendet werden, die sich von E-Mails aus Eloqua durchklicken. Führen Sie die folgenden Schritte aus, um eine Zielseite zu erstellen.

Hinweis: Vollständige Informationen zu Eloqua-Zielseiten finden Sie in der Oracle Eloqua-Dokumentation.
  1. Melden Sie sich bei Ihrem Video Cloud-Konto an. Wählen Sie das Konto mit dem Video aus, das Sie auf der Zielseite verwenden möchten. Dieses Konto muss auch Zugriff auf das Campaign-Modul haben.
  2. Melden Sie sich bei Ihrem Eloqua-Konto an.
  3. Navigieren Sie zu Vermögenswerte > Startseiten und dann klicken Erstellen Sie eine Landing Page.
  4. Klicken Leere HTML-Landingpage und dann klicken Wählen.
  5. Klicken Sie in der linken Navigation auf Cloud-Inhalt. Das Dialogfeld "Cloud-Inhalt" wird geöffnet.
  6. Zieh den Brightcove-Video Objekt auf die HTML-Seite und legen Sie es zwischen den <body> Stichworte.
    Hinweis: Einer Landing Page können mehrere Spieler hinzugefügt werden, indem Sie einen Spieler zwischen die Landing Page ziehen und dort ablegen <body> Stichworte. Speichern Sie die Landingpage, nachdem Sie jeden Spieler hinzugefügt haben.

Die Konfiguration von Cloud-Inhalten Dialog wird geöffnet, wo Sie können Wählen Sie ein Video und einen Player aus.

Video und Player auswählen

Im Dialogfeld "Konfiguration von Cloud-Inhalten" werden ein Video Cloud-Konto, ein Video und ein Player ausgewählt.

  1. Verwenden Sie die Dropdown-Liste oben rechts, um das zu verwendende Video Cloud-Konto auszuwählen. Nachdem ein Konto ausgewählt wurde, wird der Dialog aktualisiert, um die Videos und Player des ausgewählten Kontos anzuzeigen.
  2. Wählen Sie ein Video aus, das auf der Zielseite angezeigt werden soll. Die 20 neuesten Videos werden in der Liste angezeigt.
  3. Wählen Sie einen Spieler aus, der verwendet werden soll. Es werden nur kampagnenfähige Spieler angezeigt.
  4. Stellen Sie die Spielergröße.
  5. Klicken speichern um die Änderungen zu speichern.
  6. Klicken X. um das Dialogfeld "Konfiguration von Cloud-Inhalten" zu schließen. Der HTML-Code wird mit dem Code aktualisiert, um den Player einzubetten.
  7. (Optional) Fügen Sie der Zielseite weitere Objekte hinzu.
  8. Klicken speichern um die Landing Page zu speichern.
  9. Geben Sie der Zielseite einen Namen und klicken Sie dann auf speichern.
