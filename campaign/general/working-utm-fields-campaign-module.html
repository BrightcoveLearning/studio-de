UTM-Codes (auch als UTM-Parameter bezeichnet) sind einfache Textausschnitte, die Sie einer URL hinzufügen können, um den Erfolg Ihres Webinhalts zu verfolgen. Es gibt 5 Standardparameter, die Sie zu jeder URL hinzufügen können:

  • Kampagne
  • Quelle
  • Mittel
  • Kontext
  • Begriff

UTM-Codes werden üblicherweise an benutzerdefinierte URLs angehängt. Unten finden Sie ein Beispiel für UTM-Codes (fett gedruckt), die einer URL hinzugefügt wurden.

https://www.brightcove.com/en/blog/2018/08/define-your-video-strategy-3-simple-steps-1?utm_campaign=blogpost&utm_medium=social&utm_source=LinkedIn

Mithilfe von UTM-Codes können Sie verfolgen, wie ein Blog-Beitrag, eine Internet-Anzeige oder eine E-Mail-Explosion funktioniert, ohne für jede Kampagne benutzerdefinierte Zielseiten erstellen zu müssen. Durch Erstellen eines eindeutigen UTM-Codes für Blog-Posts, Anzeigen und E-Mails können Sie Daten abrufen, die mehr Verkehr, Conversions usw. generieren.

Konfigurieren von Datenverbindungen für die Arbeit mit UTM-Feldern

Campaign parst die UTM-Felder aus den URL-Argumenten. Auf die UTM-Daten kann dann über Ihre Marketing-Automatisierungssoftware oder über die Audience-API zugegriffen werden. Die erforderliche Campaign-Konfiguration basiert auf der verwendeten Marketing-Automation-Plattform.

  • HubSpot
  • Zwangsversteigerung
  • Eloqua
  • Marketo

HubSpot

Bei HubSpot-Datenverbindungen ist kein zusätzliches Setup erforderlich. Die UTM-Felder werden automatisch angezeigt.

Zwangsversteigerung

Verwenden Sie bei Salesforce-Datenverbindungen a Zuordnungstyp von Fortgeschrittene Anschließend können die UTM-Felder den entsprechenden Datenobjekten in Salesforce zugeordnet werden.

Eloqua

Bei Eloqua-Datenverbindungen ist keine zusätzliche Einrichtung erforderlich. Die UTM-Felder werden automatisch angezeigt.

Marketo

Aufgrund der Tatsache, dass nicht genügend Felder zur benutzerdefinierten Aktivität hinzugefügt werden können, die wir in Marketo erstellen, werden UTM-Felder bei Marketo-Verbindungen nicht unterstützt.

Exportieren von Daten mit dem Campaign-Modul

UTM-Felder werden beim Exportieren von Kampagnendaten und beim Auswählen von . eingeschlossen Ereignisse anzeigen. Informationen zum Exportieren von Daten finden Sie unter Exportieren von Kampagnendaten aus dem Campaign-Modul.

Zugriff auf UTM-Felder über die Audience-API

Auf UTM-Felder kann über die Audience-API zugegriffen werden. Weitere Informationen finden Sie unter Audience API-Dokumentation.
