Wenn Sie Kampagnen-Tracking oder SSO-Integrationen verwenden, können Sie mit dem Campaign-Modul ein Video auswählen und dann die Aktivität für dieses Video anzeigen. Auf diese Weise können Sie sehen, wer ein bestimmtes Video angesehen hat.

Alle Videoaufrufe in kampagnenaktivierten Playern werden erfasst. Klicken Sie auf, um die Videoaktivität anzuzeigen Video (nicht MAP) in der linken Navigation. Auf der Seite "Anzeigeaktivität" wird eine Liste aller angezeigten Videos aus SSO- und Kampagnen-Tracking-Integrationen für den ausgewählten Zeitraum angezeigt. Das Wählen Sie Verbindung Mit der Steuerung kann eine bestimmte Verbindung ausgewählt werden. Mit dem Kalendersteuerelement können Sie einen neuen Datumsbereich auswählen. Daten können durch Klicken auf die Spaltenüberschriften sortiert werden.

Anmerkungen:
  • Es wird einige Stunden dauern, bis die Anzeigedaten angezeigt werden.
  • Analyse- und Kampagnenverfolgung über unabhängige Plugins und Backend-Systeme. Es ist sehr häufig, dass nicht dieselben Ergebnisse erzielt werden. Campaign hält sich auch daran, Einstellungen im Browser nicht zu verfolgen. Die vollständigsten Analysen finden Sie in den Berichten des Analytics-Moduls.

Wenn Sie auf den Link für ein Video klicken, werden alle Anzeigeaktivitäten für dieses Video angezeigt. Daten können durch Klicken auf die Spaltenüberschriften sortiert werden.

Die Daten können durch Klicken auf exportiert werden CSV oder XLS Schaltflächen oben auf der Seite.

Andere Optionen zur Datenanzeige

Das Campaign-Modul bietet mehrere Möglichkeiten zum Anzeigen/Exportieren von Daten:

  • Letzte Aktivität - Wird verwendet, um Ereignisdaten der Videoansicht der letzten 24 Stunden anzuzeigen. Informationen finden Sie unter Anzeigen von Spielerereignissen im Kampagnenmodul.
  • Viewer-Profile - Dient zum Anzeigen der Videoaktivität durch den Betrachter. Weitere Informationen finden Sie unter Anzeigen von Viewer-Profilinformationen.
  • Export von Daten - Wird zum Exportieren von Videoansichten und Lead-Daten verwendet. Informationen finden Sie unter Exportieren von Kampagnendaten aus dem Campaign-Modul.
  • Zielgruppen-API - Die Audience-API kann zum programmgesteuerten Abrufen von Videoansichten und Lead-Daten verwendet werden. Weitere Informationen finden Sie unter Überblick: Zielgruppen-API.
