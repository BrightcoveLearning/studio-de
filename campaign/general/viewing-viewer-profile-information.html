Bei der Verwendung von Campaign-Tracking-Integrationen werden Zuschauerprofile basierend auf den Daten erstellt, die zur Identifizierung des Zuschauers verwendet wurden. Mit Viewer-Profilen können Sie die Anzeigedaten für einen bestimmten Viewer anzeigen.

Alle Videoaufrufe in kampagnenaktivierten Playern werden erfasst. Klicken Sie auf, um die Videoaktivität anzuzeigen Viewer-Profil (nicht MAP) in der linken Navigation. Auf der Seite "Viewer-Profile" wird eine Liste aller Anzeigeaktivitäten aus Kampagnen-Tracking-Integrationen für den ausgewählten Zeitraum angezeigt. Mit dem Kalendersteuerelement können Sie einen neuen Datumsbereich auswählen. Der Viewer-Wert ist der Wert, mit dem der Benutzer identifiziert wurde. In der Regel ist dies die E-Mail-Adresse, die auf einem Lead-Formular erfasst wurde. Daten können durch Klicken auf die Spaltenüberschriften sortiert werden.

Hinweis: Es wird einige Stunden Verzögerung geben, bevor die Anzeigedaten erscheinen.

Wenn Sie auf den Link für einen Viewer klicken, werden alle Anzeigeaktivitäten für diesen Viewer angezeigt. Daten können durch Klicken auf die Spaltenüberschriften sortiert werden.

Die Daten können durch Klicken auf exportiert werden CSV oder XLS Schaltflächen oben auf der Seite.

Andere Optionen zur Datenanzeige

Kampagnendaten können auf andere Weise angezeigt/exportiert werden:

  • Letzte Aktivität - Wird verwendet, um Ereignisdaten der Videoansicht der letzten 24 Stunden anzuzeigen. Informationen finden Sie unter Anzeigen von Spielerereignissen im Kampagnenmodul.
  • Videoaktivität - Wird verwendet, um die Viewer-Aktivität per Video anzuzeigen. Weitere Informationen finden Sie unter Anzeigen von Videoaktivitäten.
  • Export von Daten - Wird zum Exportieren von Videoansichten und Lead-Daten verwendet. Informationen finden Sie unter Exportieren von Kampagnendaten aus dem Campaign-Modul.
  • Zielgruppen-API - Die Audience-API kann zum programmgesteuerten Abrufen von Videoansichten und Lead-Daten verwendet werden. Weitere Informationen finden Sie unter Überblick: Zielgruppen-API.
