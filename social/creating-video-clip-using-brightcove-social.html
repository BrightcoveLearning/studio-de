Brightcove Social bietet die Möglichkeit, Videoclips aus vorhandenen Videos in Ihrem Video Cloud-Konto zu erstellen. Es gibt mehrere Gründe, warum Sie einen Clip erstellen möchten:

  • So erstellen Sie einen kurzen Teaser-Clip, der in sozialen Medien veröffentlicht werden kann
  • So begrenzen Sie die Länge eines Videos, das Sie veröffentlichen möchten (die meisten Plattformen begrenzen die Länge von Videos)
  • Sie haben festgestellt, dass kürzere Videos in sozialen Medien eine bessere Leistung erzielen

Clip erstellen

Nachdem ein Clip gespeichert wurde, wird er als Video in Ihrem Video Cloud-Konto gespeichert. Befolgen Sie diese Schritte, um einen Clip zu erstellen.
  1. Öffnen Sie Brightcove Social.
  2. Klicken Videos in der linken Navigation. Eine Liste der Videos in Ihrem Video Cloud-Konto wird angezeigt.
  3. Klicken Sie auf das Video, um einen Clip zu erstellen, und klicken Sie dann auf Clip erstellen. Der Clip-Editor wird geöffnet.
    • Um einen Clip zu erstellen, muss für das Quellvideo mindestens eine MP4-Wiedergabe verfügbar sein. Wenn die Clip erstellen Wenn die Schaltfläche deaktiviert ist, ist für das Video möglicherweise keine MP4-Wiedergabe verfügbar.
    • Sie können keinen Clip aus einem anderen Clip erstellen. Wenn Sie ein abgeschnittenes Video ausgewählt haben, wird die Clip erstellen Schaltfläche wird deaktiviert.
    • Sie können keinen Clip aus einem Video erstellen, für das DRM aktiviert ist (die Sperre wird in der Miniaturansicht des Videos angezeigt) und das mithilfe eines älteren Aufnahmeprofils transkodiert wurde. DRM-fähige Videos, die mit einem Dynamic Delivery-Aufnahmeprofil transkodiert wurden, können abgeschnitten werden. Wenn ein Clip nicht erstellt werden kann, wird der Clip erstellen Schaltfläche wird deaktiviert.
    • Videos mit Untertiteln können über Social Media abgeschnitten werden, das resultierende abgeschnittene Video enthält jedoch keine Untertitel. Sie müssen neue Untertitel in das abgeschnittene Video hochladen, und Social verwendet diese beim Veröffentlichen.
  4. Geben Sie ein Clip Name.
  5. Ziehen Sie die Ziehpunkte, die auf der Timeline angezeigt werden, um den Anfang und das Ende Ihres Videos zu kürzen. Sie können auch bestimmte Start- und Endzeiten über die Bearbeitungsfelder eingeben. Eine genauere Steuerung beim Bearbeiten kann mit dem erreicht werden Tastatürkürzel. Wenn Sie fertig sind, klicken Sie auf Fortsetzen.
    Hinweis: Der Player im Clipping-Modal wird zum angezeigten Zeitpunkt auf ein oder zwei Frames des tatsächlichen Frames geschrubbt. Das Verhalten ist das gleiche, unabhängig davon, ob Sie die Schieberegler mit der Maus ziehen oder die Clip-Punkte mit den Tastenkombinationen festlegen. Der Clip-Editor schneidet genau zu den bereitgestellten Zeiten ab.
  6. Überprüfen Sie die Clip-Informationen. Der Clip-Name, die Tags und alle benutzerdefinierten Felder, die für das Konto erstellt wurden, können bearbeitet werden.
    Hinweis: Möglicherweise möchten Sie Tags entfernen, die dazu führen, dass der Clip in Ihren vorhandenen Wiedergabelisten angezeigt wird oder automatisch mit einem sozialen Ziel synchronisiert wird.
  7. Wenn Sie fertig sind, wählen Sie eine Option:
    • Speichern - Startet mit der Transcodierung und speichert den Clip dann in Ihrem Video Cloud-Konto.
    • Speichern und veröffentlichen - Startet die Transcodierung und speichert den Clip in Ihrem Video Cloud-Konto. Sie können auch die Ziele auswählen, an denen das Video veröffentlicht werden soll. Es kann eine Verzögerung geben, bis veröffentlichte Clips auf Ihren Social Media-Websites angezeigt werden, da das Video transkodiert werden muss.
    • Stornieren - Verwirft den Clip.

Gespeicherte Clips werden mithilfe des als Standardeinstellung für Social Clipping für das Konto. Informationen zum Festlegen von Standard-Aufnahmeprofilen finden Sie unter Verwalten von Aufnahmeprofilen.

Beachten Sie, dass beim Erstellen von Clips das gesamte Quellvideo heruntergeladen werden muss. Bei längeren Quellvideos kann es länger dauern, bis der Clip in Ihrem Video Cloud-Konto verfügbar ist.

Hinweis: Der Name der Clip-Quelldatei wird auf denselben Quelldateinamen wie das Quellvideo festgelegt.

Verwenden von Tastenkombinationen

Eine genauere Steuerung des Ausschnitts kann über die Tastenkombinationen erfolgen:

Abkürzung Beschreibung
J. oder ← Zurück einen Frame
K. Spielpause
L. oder → Einen Frame weiterleiten
ICH Markieren Sie den aktuellen Frame als Punkt
Ö Markieren Sie den aktuellen Frame als Outpoint
Leertaste Spielpause
⇧ + I. Gehe zu Punkt
⇧ + O. Gehe zum Punkt
