Players Module UI - Beta aktualisiert

12. März 2018

Ein neuer Player-Editor (die Player-Detailseite) ist jetzt ab der Beta-Option verfügbar. Publishern wird auf der aktuellen Player-Detailseite eine Schaltfläche mit der Aufschrift "Zum Beta-Editor wechseln" angezeigt. Wenn Sie darauf klicken, wird ein Modal angezeigt, das erklärt, was der neue Player-Editor ist und wie Feedback abgegeben werden kann. Es können sowohl die neue als auch die alte Version der Benutzeroberfläche verwendet werden. Der Wechsel ist benutzerspezifisch, sodass Sie nicht die Benutzeroberfläche für alle Benutzer im Konto beeinflussen, wenn Sie sich für die Anmeldung entscheiden. Beachten Sie, dass die Möglichkeit zum Verwalten von Player-Plugins in der neuen Benutzeroberfläche noch nicht verfügbar ist.
