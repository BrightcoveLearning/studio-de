Die Videoverfügbarkeitseigenschaften können verwendet werden, um die Daten und Zeiten zu steuern, zu denen ein Video angezeigt werden kann. Mithilfe der Geofilterung können Sie steuern, in welchen Ländern das Video angezeigt werden kann.

Anmerkungen:
  • Das für die Terminplanung verwendete Datenformat verwendet das US-Format: MM / TT / JJJJ.
  • Die Videoverfügbarkeitseigenschaften sind nur für Enterprise- und Professional-Konten verfügbar.
  • Das Planen eines Videos hat keinen Einfluss darauf, ob ein Video aktiv oder inaktiv ist oder wird. Die Planung wirkt sich nur auf aktive Videos aus.
  • Geofilter- und Planungseigenschaften können auch mit dem hinzugefügt werden CMS API wenn du erstellen oder aktualisieren ein Video.
  • Einige Verlage haben die Möglichkeit gefordert, Videos sowohl auf US-Militärbasen als auch in die USA zu filtern. Um dies zu unterstützen, gibt es eine U.S. Militär Möglichkeit. Obwohl es keinen 100% zuverlässigen Weg gibt, eine Militärbasis zu identifizieren, wird eine IP, die sich in eine MIL-Domäne auflöst, angetroffen wird und sich nicht in den USA befindet, sie wird als von einer US-Militärbasis und nicht von ihrem tatsächlichen Herkunftsland als von ihrem tatsächlichen Herkunftsland identifiziert. quot;.

Um die Verfügbarkeitseigenschaften für ein Video anzuzeigen und zu bearbeiten, öffnen Sie das Medienmodul, klicken Sie auf den Videotitel-Link und suchen Sie das Verfügbarkeit Sektion.

Klicken Sie auf, um die Verfügbarkeitseigenschaften zu bearbeiten Bearbeiten Schaltfläche oder Doppelklick in den Abschnitt. Die Videoverfügbarkeit kann basierend auf folgenden Einstellungen konfiguriert werden:

  • Terminzeit
  • Geofilterung

Datum und Uhrzeit konfigurieren

Für Videos können ein Startdatum / eine Startzeit und ein Enddatum / eine Endzeit festgelegt werden. Die Datums- / Uhrzeitsteuerung steuert, wann ein Video zum Anzeigen verfügbar sein wird.

Standardmäßig sind Videos Sofort verfügbar und haben Kein Enddatum. So legen Sie Start- / Enddaten und -zeiten fest:

  1. Klicken Sie auf das Optionsfeld neben dem Feld Datum / Uhrzeit.
  2. Klicken Sie auf das Kalendersymbol und wählen Sie mit dem Kalendersteuerelement ein Datum aus.
  3. Klicken Sie auf das Uhrsymbol () um eine Zeit einzustellen.
    Hinweis: Die Zeitzone basiert auf der Zeitzone, die auf Ihrem Computer oder Gerät festgelegt ist.
  4. Um zum Kalendersteuerelement zurückzukehren, klicken Sie auf das Kalendersymbol ().
  5. Klicken speichern.

Geofilter konfigurieren

Mithilfe der Geofilterung können Sie steuern, welche Länder das Video abspielen können und welche nicht.

Anmerkungen:
  • Die Möglichkeit, die Geofilterung von Spielern zu verwenden, ist eine Funktion, die Enterprise- und Professional-Konten gegen eine zusätzliche Gebühr zur Verfügung steht. Wenden Sie sich an Brightcove, wenn Sie Ihr Konto um eine Geofilterung erweitern möchten.
  • Geofilterung kann auch sein auf Spieler angewendet. In der Video Cloud werden sowohl Player- als auch Video-Geofilterungsrichtlinien angewendet. Wenn ein Land von einem der beiden Länder ausgeschlossen wird, ist das Video dort nicht verfügbar.

So legen Sie die Geofiltereigenschaften für ein Video fest:

  1. Wählen Ermöglichen Sie die Anzeige aus den folgenden Ländern oder Verhindern Sie die Anzeige aus den folgenden Ländern.
  2. Geben Sie die entsprechenden Ländernamen ein. Während der Eingabe wird die Länderliste gefiltert.
  3. Klicken speichern.
