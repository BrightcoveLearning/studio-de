Ordner können innerhalb des Medienmoduls erstellt und zum Organisieren von Videos verwendet werden. Sie können Videos nach Thema, Autor, Produkt, Region usw. organisieren. Wenn Sie Videos in Ordnern haben, können sie in einem Konto mit Hunderten oder Tausenden von Videos einfacher verwaltet werden. Ordner werden im seitlichen Navigationsbereich angezeigt. Die Anzahl der Videos in jedem Ordner wird neben dem Ordnernamen angezeigt. Klicken Sie auf die seitliche Navigationsschaltfläche um das seitliche Navigationsfeld auszublenden oder anzuzeigen.

Hinweis: Verschachtelte Ordner werden nicht unterstützt.
Hinweis: Es können maximal 100 Videos gleichzeitig in einen Ordner verschoben werden.

Ordner erstellen

Gehen Sie folgendermaßen vor, um einen neuen Ordner zu erstellen:

  1. Öffne das Medien Modul.
  2. Klicken Ordner erstellen im seitlichen Navigationsbereich.
  3. Geben Sie einen Namen für den Ordner gemäß den folgenden Richtlinien ein. Drücken Sie Enter.
    • Der Name ist auf 100 Zeichen begrenzt
    • Ordnernamen müssen innerhalb eines Kontos eindeutig sein
    • Unzulässiges Zeichen: /
Hinweis: Es können maximal 1000 Ordner erstellt werden.

Um den Ordner schnell zu löschen, klicken Sie auf X. nach dem Ordnernamen.

Hinzufügen von Videos zu Ordnern

Es gibt verschiedene Möglichkeiten, Videos zu einem Ordner hinzuzufügen. Sie können:

  • Fügen Sie einem Ordner mehrere Videos hinzu
  • Bearbeiten Sie die Videoeigenschaften, um einen Ordner zuzuweisen
Hinweis: Die Anzahl der Videos, die sich in einem Ordner befinden können, ist nicht begrenzt.

Hinzufügen mehrerer Videos zu einem Ordner

Befolgen Sie diese Schritte, um einem Ordner mehrere Videos gleichzeitig hinzuzufügen.

  1. In dem Medien Modul, klicken Sie auf Alle Videos im seitlichen Navigationsbereich, um alle Videos anzuzeigen.
    Hinweis: Die Zahl nach dem Ordnernamen gibt an, wie viele Videos sich in diesem Ordner befinden.
  2. Wählen Sie die Videos aus, die einem Ordner hinzugefügt werden sollen, indem Sie auf das Kontrollkästchen neben der Video-Miniaturansicht klicken. Klicken Mehr> In Ordner verschieben.
  3. Wählen Sie einen Ordner und klicken Sie auf Bewegung.

Bearbeiten der Videoeigenschaften, um einen Ordner zuzuweisen

Videos können einem Ordner hinzugefügt werden, indem Sie die Videoeigenschaften bearbeiten und einen Ordner zuweisen.

  1. In dem Medien Modul, klicken Sie auf den Link für das Video.
  2. In dem Kategorisierung Abschnitt, wählen Sie a Mappe. Sie können auch einen neuen Ordner erstellen, indem Sie auf klicken + Ordner erstellen.

Anzeigen von Videos in einem Ordner

Um die Videos in einem Ordner anzuzeigen, klicken Sie auf einen Ordnernamen.

Videos aus Ordnern entfernen

Sie können Entfernen Sie mehrere Videos aus einem Ordner oder entfernen Sie ein einzelnes Video aus einem Ordner von Bearbeiten der Videoeigenschaften.

Entfernen mehrerer Videos aus einem Ordner

Mit diesen Schritten können mehrere Videos aus einem Ordner entfernt werden.

  1. In dem Medien Modul, klicken Sie auf den Link für den Ordner.
  2. Aktivieren Sie die Kontrollkästchen neben den Videos, um sie zu entfernen und zu klicken Videos entfernen> Videos aus Ordner entfernen.

Bearbeiten der Videoeigenschaften

Videos können aus einem Ordner entfernt werden, indem Sie die Videoeigenschaften bearbeiten und den Ordner im Bereich "Videoinformationen" entfernen.

  1. In dem Medien Modul, klicken Sie auf den Link für das Video.
  2. In dem Kategorisierung Sektion, set the Mappe zu --- --- ..

Beachten Sie, dass diese Methode auch verwendet werden kann, um einen anderen Ordner auszuwählen, in den das Video verschoben werden soll.

Ordner löschen und umbenennen

Ordner können folgendermaßen umbenannt oder gelöscht werden:

  1. Klicken Sie auf den Ordner, der umbenannt oder gelöscht werden soll. Klicken Sie auf den Abwärtspfeil neben dem Ordner und klicken Sie auf Umbenennen oder Löschen.
