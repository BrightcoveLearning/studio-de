Die Schnellbearbeitungsfunktion bietet eine bequeme Möglichkeit, Videosigenschaften für mehrere Videos zu bearbeiten. Sie können mehrere Videos auswählen, indem Sie auf ein Kontrollkästchen klicken und dann bei gedrückter Umschalttaste auf ein anderes Kontrollkästchen klicken (halten Sie die Umschalttaste gedrückt, während Sie auf das zweite Video klicken, und die beiden Videos und alle dazwischen liegenden Videos werden ausgewählt). Klicken Sie auf Schnelle Bearbeitung öffnet das Schnellbearbeitungsfenster auf der rechten Seite des Medienmoduls.

Im Schnellbearbeitungsfenster wird eine Teilmenge der Videoeigenschaften angezeigt:

  • Name
  • Kurze Beschreibung
  • Lange Beschreibung
  • Tags zum Hinzufügen
  • Zu entfernende Tags
  • Verwandte Link-URL
  • Zugehöriger Link-Text
  • Werbung
  • Anzeigenschlüssel
  • Planen von Start- / Enddaten
  • Geofilterung
  • Benutzerdefinierte Felder
  • Wiedergabebeschränkungen
Hinweis: Eine Beschreibung all dieser Eigenschaften finden Sie im Dokument Bearbeiten von Videoeigenschaften mit dem Medienmodul.

Verwenden Sie das Schnellbearbeitungsfenster, um Werte zuzuweisen. Wenn Sie fertig sind, klicken Sie auf Speichern, um die Änderungen zu speichern, oder auf Abbrechen, um das Programm zu beenden.

Hinweise zur Verwendung von Quick Edit

  • Wenn mindestens einem Video Werte zugewiesen sind, die einer Eigenschaft zugewiesen sind, Gemischten Wert bearbeiten wird Angezeigt werden
  • Ein Sternchen (*) nach dem Eigenschaftsnamen zeigt eine erforderliche Eigenschaft an
  • Klicken Sie auf, um an einer Eigenschaft vorgenommene Änderungen abzubrechen Wiederherstellen
