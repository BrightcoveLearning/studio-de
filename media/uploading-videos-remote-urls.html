Das Upload-Modul verwendet einen dynamischen Aufnahmeprozess, um Videoinhalte abzurufen und zu transkodieren. Wenn Sie eine Datei zum Hochladen auswählen, verwendet Video Cloud einen Pull-basierten Prozess, um den Inhalt von einem Speicherort abzurufen, an dem der Inhalt mithilfe des Zencoder-Transcodierungsdienstes transkodiert wird.

Das Upload-Modul unterstützt verschiedene Methoden zum Hochladen von Videos:

  • Drag & Drop - Mit dieser Methode können Sie Dateien aus Ihrem lokalen Dateisystem auf die Seite Upload-Modul ziehen
  • Dialogfeld zur Dateiauswahl - Diese Methode öffnet einen Dateiauswahldialog, in dem Sie Dateien zum Hochladen durchsuchen und auswählen
  • Remote-URL - Mit dieser Methode können Sie eine URL eingeben, um den Speicherort der Quelldatei anzugeben

Dieses Thema konzentriert sich darauf, wie Sie die Remote-URL für beliebte Cloud-Datei-Hosting-Dienste abrufen können. Die Verwendung der Remote-URL-Upload-Methode ist hilfreich für größere Dateien, die möglicherweise von Personen außerhalb Ihrer Organisation bearbeitet / verarbeitet wurden. Anstatt eine große Datei auf Ihren lokalen Computer zu kopieren, kann eine Remote-URL zur Datei bereitgestellt werden, und Video Cloud kann die Datei vom Speicherort abrufen. In jedem Fall müssen Sie sicherstellen, dass die URL zur Datei öffentlich verfügbar ist, da sonst der Upload fehlschlägt.

Hinweis: Ausführliche Informationen zur Verwendung des Upload-Moduls finden Sie unter Hochladen von Videos mit dem Upload-Modul.
Hinweis: Stellen Sie vor dem Hochladen von Videos sicher, dass das entsprechende Aufgabeprofil ausgewählt wurde.

Google Drive

Gehen Sie folgendermaßen vor, um die URL für ein in Google Drive gespeichertes Video abzurufen:

  1. Melden Sie sich bei Ihrem Google Drive-Konto an.
  2. Suchen Sie das Video, das Sie hochladen möchten, und klicken Sie mit der rechten Maustaste darauf.
  3. Klicken Get Shareable Link.
  4. Stellen Sie für die Berechtigungen sicher, dass Sie die Option so auswählen Jeder mit dem Link kann auf die Datei zugreifen. Wenn die Berechtigung auf gesetzt ist Beschränkt Der Upload schlägt fehl.
    Hinweis: Wenn Sie über ein Google-Unternehmenskonto verfügen, sind Ihre einzigen Freigabeoptionen möglicherweise eingeschränkt oder für Ihre Domain freigegeben. Wenn Sie nicht in der Lage sind, die Datei mit dem Link an Jedermann freizugeben, kann Video Cloud nicht auf die Datei zugreifen und der Upload schlägt fehl.
  5. Klicken Link kopieren. Dies erzeugt leider keinen direkten Link zur Videodatei. Wir können eine andere Site verwenden, um über den Freigabelink einen direkten Link zur Videodatei zu generieren.
  6. Navigieren Sie zu https://sites.google.com/site/gdocs2direct/home. Es sind auch andere Websites verfügbar, die einen direkten Link zu Google Drive-Dateien generieren.
  7. Fügen Sie Ihre Freigabe-URL ein.
  8. Klicken Direkten Link erstellen.

Die im Feld "Output-Link" zurückgegebene URL ist die URL, die im Upload-Modul zum Hochladen des Videos verwendet werden soll.

Hinweis: Sie können keine YouTube-URL verwenden, da keine direkten Links zu Videodateien generiert werden.
Hinweis: Uploads funktionieren nicht, wenn eine Größenwarnung für Google Drive angezeigt wird (normalerweise für Dateien> 100MB). Selbst nach dem Ändern der URL wird sie auf eine Größenfehlerseite umgeleitet und der Upload schlägt fehl.

Dropbox

Gehen Sie folgendermaßen vor, um die URL für ein in Dropbox gespeichertes Video abzurufen:

  1. Melden Sie sich bei Ihrem Dropbox-Konto an.
  2. Suchen Sie das Video, das Sie hochladen möchten, und klicken Sie auf Teilen Verknüpfung.
  3. Klicken Verknüpfung erstellen um einen Link zur Datei zu erstellen.
  4. Klicken Link kopieren um den Link in die Datei zu kopieren.

Beachten Sie, dass die URL zur Datei möglicherweise mit endet ?dl=0. Um den Browser zum Herunterladen des Inhalts des Links zu zwingen, müssen Sie diesen Teil der URL in ändern ?dl=1.

Box

Gehen Sie folgendermaßen vor, um die URL für ein in Box gespeichertes Video abzurufen:

Hinweis: Die Möglichkeit, einen direkten Link zu einer Datei zu erhalten, ist eine Box-Premium-Funktion, die für kostenlose persönliche Konten nicht verfügbar ist.
  1. Melden Sie sich bei Ihrem Box-Konto an.
  2. Suchen Sie das Video, das Sie hochladen möchten, und klicken Sie auf Teilen Verknüpfung.
  3. Drücke den Erstellen Sie einen gemeinsamen Link Schieberegler, um einen Link zur Datei zu erstellen. Dadurch wird ein Vorschau-Link zur Datei erstellt.
  4. Klicken Linkeinstellungen. Sie müssen der Datei einen direkten Download-Link zuweisen.
  5. Klicken Kopieren um den Link in die Zwischenablage zu kopieren.

Andere Dienstleistungen

Es sollte keine Probleme bei der Verwendung von URLs für Videodateien geben, die auf anderen Cloud-Plattformen (z. B. Amazon S3) gehostet werden, solange die URL a ist öffentlich zugängliche URL.

Hinweis: Beim Aufnehmen von Videodateien aus Amazon S3 kann eine Bucket-Richtlinie erstellt werden, damit auf Videos, die nicht für alle verfügbar sind, über die Video Cloud zugegriffen werden kann. Weitere Informationen finden Sie im Entwicklerdokument Using Dynamic Ingest with S3.
